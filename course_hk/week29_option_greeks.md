# 第29週：期權希臘字母指標——衡量與管理風險

## 閱讀部分

### a) 為何這一課至關重要

在第25至28週，你學習了如何運用期權作為工具：買入認購期權和認沽期權、賣出備兌認購期權，以及賣出現金擔保認沽期權。你已掌握期權運作的基本機制。但懂得駕駛汽車，和看懂儀表盤是兩回事。希臘字母指標就是你的儀表盤。它們精確告訴你，當市場狀況改變時，你的期權倉位將如何反應。

**為何每位期權交易者都必須掌握希臘字母指標：**

**1. 沒有希臘字母指標，你等同盲目飛行。** 假設你賣出一張備兌認購期權，股票上升了3美元。你的認購期權虧損了多少？隔夜時間流逝令它獲益了多少？如果隱含波動率下跌2%，它又會有多大變化？沒有希臘字母指標，你只是在猜測。有了希臘字母指標，你可以精確回答每一個問題。專業期權交易者在建立倉位前，必定清楚自己的希臘字母風險敞口。你亦應如此。

**2. 希臘字母指標解釋你的倉位為何盈虧。** 許多初學期權的交易者感到困惑：為何股票上升，認購期權卻未有跟隨上漲？或者股票下跌，認沽期權的價值卻反而縮水？希臘字母指標解釋了這些表面上的矛盾。也許是波動率崩塌抵消了方向性的盈利；也許是時間值消耗蠶食了期權金。掌握希臘字母指標，就能化迷惑為清晰。

**3. 希臘字母指標讓你在風險成為問題之前提早管理。** 如果你知道自己的投資組合德爾塔為+500，你便知道市場每上升1美元，你大約獲利500美元；市場每下跌1美元，你大約虧損500美元。若這個風險敞口過大，你可以在市場波動前先行調整，而非事後補救。風險管理正是專業交易與賭博之間的分水嶺。

**4. 希臘字母指標貫通後續所有章節。** 第30週（價差與鐵鷹式策略）、第37週（期權槓桿）、第38週（長期期權）及第40週（波動率指數與波動率）均預設你能以希臘字母指標思考問題。價差策略從根本上說，是關於如何組合不同的希臘字母風險敞口。不理解維加，波動率交易便無從談起。長期期權的分析，有賴於理解德爾塔和西塔在長存續期下的表現。本週是所有進階期權內容的基礎。

**5. 希臘字母指標讓你掌握市場的共同語言。** 當財經媒體提到「伽馬軋空」或「維加風險」，你將確切明白其含義。當你的證券行平台顯示希臘字母數值時，你將知道如何解讀。你將能夠流利運用期權的專業語言。

我們將涵蓋全部五個主要希臘字母指標：德爾塔、伽馬、西塔、維加和羅。我們亦會探討它們如何相互作用，以及如何將其應用於實際的倉位管理。學完本課，你將能夠審視任何期權倉位，清楚了解自己所承擔的風險，以及倉位在市場波動下的表現。

### b) 你需要掌握的知識

#### 希臘字母指標概覽

希臘字母指標是敏感性衡量工具。每個希臘字母告訴你，當某一特定變量改變而其他條件不變時，期權價格會變動多少。對學過微積分的人來說，它們是偏導數，但你無需懂微積分，同樣能有效運用它們。

```
五大主要希臘字母指標：

  指標    衡量對象的敏感性     範圍（買入認購期權）    類比
  ====    ================    ==================    ========
  德爾塔  股票價格變動          0 至 +1.0             車速表
  伽馬    德爾塔的變動          永遠為正              加速度
  西塔    時間流逝              永遠為負              融化的冰
  維加    隱含波動率變動        永遠為正              風向標
  羅      利率變動              小幅正值              恆溫器

  注意：以上為買入認購期權的情況。沽空倉位符號相反，
  認沽期權與認購期權亦有部分符號相異（詳見以下各節）。
```

可以這樣理解：如果期權是一輛汽車，德爾塔告訴你行駛速度，伽馬告訴你速度變化的快慢（加速度），西塔告訴你每小時消耗多少燃料（時間成本），維加告訴你對路面狀況有多敏感（波動性），而羅則告訴你氣溫對行駛有多大影響（利率，通常影響較小）。

#### 德爾塔：方向性風險敞口

對大多數投資者而言，德爾塔是最重要的希臘字母指標。它衡量當相關股票價格每變動1美元時，期權價格會相應變動多少。

```
德爾塔數值：

  買入認購期權：  德爾塔範圍為 0 至 +1.0
  買入認沽期權：  德爾塔範圍為 -1.0 至 0
  賣出認購期權：  德爾塔範圍為 -1.0 至 0
  賣出認沽期權：  德爾塔範圍為 0 至 +1.0

  股票（供參考）：
  持有100股多倉  = 德爾塔 +100（即每股 +1.0）
  持有100股空倉  = 德爾塔 -100（即每股 -1.0）
```

**示例：** 若一張認購期權的德爾塔為0.60，當股票上升1美元，期權大約升值0.60美元；當股票下跌1美元，期權大約跌值0.60美元。由於一張合約控制100股，每當股票移動1美元，合約約盈虧60美元。

```
德爾塔與價值狀態的關係：

  股票價格 = $150

  深度價內認購期權（行使價$120）：  德爾塔 ~ 0.95
  價內認購期權（行使價$140）：      德爾塔 ~ 0.75
  平價認購期權（行使價$150）：      德爾塔 ~ 0.50
  價外認購期權（行使價$160）：      德爾塔 ~ 0.30
  深度價外認購期權（行使價$180）：  德爾塔 ~ 0.05

  認購期權德爾塔曲線：

  德爾塔
  1.0 |                                      ___________
      |                                  ___/
  0.8 |                              ___/
      |                          ___/
  0.6 |                      ___/
      |                  ___/
  0.4 |              ___/
      |          ___/
  0.2 |      ___/
      |  ___/
  0.0 |_/
      +----+----+----+----+----+----+----+----+----+----
       80   90  100  110  120  130  140  150  160  170
                         股票價格
                    （行使價 = $130）
```

**德爾塔作為到期實值概率的近似值：** 德爾塔大致近似於期權在到期時處於價內的概率。德爾塔0.30意味著大約有30%的機會在到期時處於價內。這在數學上並不精確（德爾塔與到期實值概率的計算方式不同），但作為快速估算的心理捷徑相當實用。

**德爾塔作為等值持股量：** 德爾塔告訴你期權的行為等同於持有多少股股票。持有德爾塔為0.50的一張認購期權，大致等同於持有50股股票。這個概念稱為「德爾塔等值倉位」。

```
德爾塔等值倉位：

  倉位                            德爾塔    等值於
  ==============================  ======    ======================
  買入1張平價認購期權（德爾塔0.50） +50      持有50股多倉
  買入2張平價認購期權（德爾塔0.50） +100     持有100股多倉
  買入1張價內認購期權（德爾塔0.80） +80      持有80股多倉
  賣出1張平價認沽期權（德爾塔-0.50）+50      持有50股多倉
  備兌認購期權（持股+賣出平價認購期權： +30  持有30股多倉
    +100 + (-70)）
```

#### 伽馬：德爾塔變動速率

伽馬衡量當股票價格每移動1美元時，德爾塔變動多少。若德爾塔是速度，伽馬便是加速度。伽馬告訴你方向性風險敞口的變化有多快。

```
伽馬特性：

  伽馬在以下情況最高：
    - 平價期權（德爾塔 ~ 0.50）
    - 臨近到期的期權（少於7天）
    - 低波動率環境

  伽馬在以下情況最低：
    - 深度價內或深度價外期權
    - 距到期日較遠的期權（超過90天）
    - 高波動率環境

  按價值狀態劃分的伽馬輪廓（距到期日30天）：

  伽馬
  0.05 |
       |           *****
  0.04 |        ***     ***
       |      **           **
  0.03 |    **               **
       |   *                   *
  0.02 |  *                     *
       | *                       *
  0.01 |*                         *
       |                           *
  0.00 +----+----+----+----+----+----
       80   90  100  110  120  130  140
                 股票價格
              （行使價 = $110）
```

**伽馬為何重要：**

當你持有長伽馬倉位（買入期權），德爾塔朝有利方向移動。若股票上漲，你的德爾塔增加，你在進一步上行時獲益更多。若股票下跌，你的德爾塔減少，你在進一步下行時的損失較少。長伽馬猶如內置的自動調節機制。

當你持有短伽馬倉位（賣出期權），德爾塔朝不利方向移動。若股票上漲，你的短倉認購期權的負德爾塔愈來愈大（你相當於愈來愈沽空一隻上升中的股票）。若股票下跌，你的短倉認沽期權的正德爾塔愈來愈大（你相當於愈來愈多地持有一隻下跌中的股票多倉）。短伽馬是期權賣方為換取期權金收入所承擔的風險。

```
伽馬對德爾塔的影響：

  示例：AAPL平價認購期權，行使價$150，德爾塔 = 0.50，伽馬 = 0.03

  股票上升$1至$151：
    新德爾塔 = 0.50 + 0.03 = 0.53
    期權現在每上升$1，額外獲益$0.53

  股票下跌$1至$149：
    新德爾塔 = 0.50 - 0.03 = 0.47
    期權現在每下跌$1，額外損失$0.47

  股票上升$5至$155：
    德爾塔從0.50移向~0.65（每升$1，伽馬累加~0.03）
    期權在上升途中表現愈來愈像股票

  股票下跌$5至$145：
    德爾塔從0.50移向~0.35
    期權在下跌途中的敏感度愈來愈低

  這正是長伽馬的魅力所在：在盈利時加速，在虧損時減速。
```

**臨近到期時的伽馬風險：**

對於平價期權，伽馬在臨近到期時急劇放大。這正是到期週對期權賣方而言極為危險的原因。距到期日僅一天的平價期權，其伽馬可能高達0.15甚至更高，意味著德爾塔會隨股票小幅波動而劇烈搖擺。

```
按距到期日天數劃分的伽馬（平價期權）：

  距到期日天數  伽馬      詮釋
  ===========  =====    ====================================
  90天         0.012    德爾塔變動緩慢，倉位穩定
  60天         0.015    德爾塔敏感度適中
  30天         0.022    德爾塔反應開始增強
  14天         0.035    價格大幅波動時德爾塔明顯擺動
  7天          0.055    德爾塔相當跳躍
  3天          0.090    德爾塔大幅擺動
  1天          0.180    德爾塔極不穩定
  到期日       極大      平價期權德爾塔在0與1.0之間翻轉
```

#### 西塔：時間值消耗

西塔衡量在其他條件不變的情況下，每過一天期權價格會下跌多少。我們在第25週已介紹過這個概念，現在可以精確量化。

```
西塔特性：

  買入期權：  西塔為負值（每天虧損時間值）
  賣出期權：  西塔為正值（每天獲得時間值收益）

  西塔在以下情況最大（負值最大）：
    - 平價期權
    - 臨近到期的期權
    - 高隱含波動率期權

  西塔消耗曲線（平價期權，初始期權金$5.00）：

  期權金
  $5 |*
     | *
  $4 |  *
     |   *
  $3 |    **
     |      **
  $2 |        ***
     |           ***
  $1 |              *****
     |                   *********
  $0 +----+----+----+----+----+----
     90   75   60   45   30   15   0
              距到期日天數
```

**西塔作為每日成本：**

若西塔為-$0.05，期權每股每日損失0.05美元，即每張合約每日損失5.00美元。週末兩天（非交易日）合計損失約10.00美元。

```
按行使價與到期日劃分的西塔（股票報價$150）：

  距到期日30天：
  行使價  類型  期權金    每日西塔    損失50%所需天數
  ======  ====  =======  =========   ================
  $140    認購  $12.50   -$0.03      ~208天（深度價內）
  $145    認購  $8.20    -$0.05      ~82天
  $150    認購  $5.00    -$0.08      ~31天
  $155    認購  $2.80    -$0.07      ~20天
  $160    認購  $1.20    -$0.04      ~15天

  注意：平價期權的西塔絕對值最高，但價外期權每日流失的
  期權金比例更高。

  距到期日7天：
  行使價  類型  期權金   每日西塔    每日流失比例
  ======  ====  =======  =========  ==========
  $145    認購  $5.80    -$0.04      0.7%
  $150    認購  $1.80    -$0.18      10.0%
  $155    認購  $0.40    -$0.10      25.0%

  在最後一週，平價期權的西塔是距到期日30天時的三倍。
```

**西塔對策略選擇的重要性：**

當你賣出期權（備兌認購期權、現金擔保認沽期權），西塔是你的盈利引擎。你希望西塔最大化，這意味著賣出距到期日30至45天的平價或輕微價外期權。當你買入期權，西塔是你的持倉成本。你希望通過買入較長存續期或價內期權（內在值比例較高，外在值消耗較少）來將其最小化。

#### 維加：波動率敏感性

維加衡量當隱含波動率每變動一個百分點時，期權價格會變動多少。與德爾塔和西塔不同，維加對認購期權和認沽期權的符號相同。

```
維加特性：

  買入期權（認購或認沽）：  維加為正值
    -> 隱含波動率上升時獲益，下跌時虧損

  賣出期權（認購或認沽）：  維加為負值
    -> 隱含波動率下跌時獲益，上升時虧損

  維加在以下情況最大：
    - 平價期權
    - 距到期日較長的期權（與西塔相反！）
    - 高價股票（以絕對金額而言）
```

```
維加示例：

  AAPL $150認購期權，距到期日30天
  當前隱含波動率 = 25%，期權金 = $4.00，維加 = $0.10

  若隱含波動率從25%升至27%（+2個百分點）：
    期權金變動 = +2 x $0.10 = +$0.20
    新期權金 = $4.20

  若隱含波動率從25%跌至22%（-3個百分點）：
    期權金變動 = -3 x $0.10 = -$0.30
    新期權金 = $3.70

  按合約計算（100股）：
    隱含波動率升2個百分點：獲益$20
    隱含波動率跌3個百分點：虧損$30
```

**波動率崩塌：** 這正是維加至關重要的原因。在業績公告前，隱含波動率因市場預期大幅波動而急升。公告後，不確定性消除，隱含波動率迅速回落至正常水平。這被稱為「波動率崩塌」或「隱含波動率崩塌」。

```
波動率崩塌示例：

  AAPL $150認購期權，業績公告前7天：
    隱含波動率 = 45%，期權金 = $6.00，維加 = $0.08

  AAPL公佈業績。股票從$150升至$153（+$3）。
  但隱含波動率從45%跌至28%（-17個百分點）。

  德爾塔盈利：  +$3 x 德爾塔0.50 = +$1.50
  維加虧損：    -17 x 維加$0.08  = -$1.36
  西塔虧損：    -1天 x $0.15     = -$0.15

  淨變動：+$1.50 - $1.36 - $0.15 = -$0.01

  股票上升了$3，認購期權卻幾近打和。

  這就是維加的威力。許多初學者在業績前買入認購期權，
  方向判斷正確，卻仍然虧損，原因就是沒有計入波動率崩塌。
```

```
按距到期日天數劃分的維加（平價期權，股票報價$150）：

  距到期日天數  每1%隱含波動率的維加  隱含波動率25%時的期權金
  ===========  ==================    ====================
  7天          $0.04                 $2.10
  14天         $0.06                 $3.00
  30天         $0.10                 $4.30
  60天         $0.14                 $6.10
  90天         $0.17                 $7.50
  180天        $0.24                 $10.60
  365天        $0.34                 $15.00

  重要啟示：距到期日越長的期權，維加越大。
  長期期權對隱含波動率變化極為敏感。
  短期期權維加較小，但伽馬較大。
```

#### 羅：利率敏感性

羅衡量當利率每變動一個百分點時，期權價格會變動多少。對大多數散戶投資者而言，羅是最不重要的希臘字母指標，因為利率變動緩慢，且對短期期權的影響甚微。

```
羅的特性：

  買入認購期權：  羅為正值（利率上升 -> 認購期權價格上升）
  買入認沽期權：  羅為負值（利率上升 -> 認沽期權價格下跌）

  原因？利率上升增加了持有股票的融資成本。
  認購期權是買入股票的替代品，當替代選擇（買股票）的
  融資成本更高時，認購期權的價值便上升。

  羅的示例：

  AAPL $150認購期權，距到期日90天，羅 = $0.08

  若利率上升1%（例如從4.5%升至5.5%）：
    期權金變動 = +每股$0.08（每張合約+$8）

  對於長期期權（365天以上），羅可達$0.30至$0.50
  對於每週期權，羅接近於零。

  羅在以下情況值得關注：
  - 在利率變動周期內持有的長期期權倉位
  - 持有大量期權風險敞口的大型投資組合
  - 聯儲局快速調息期間（如2022至2023年）
```

#### 希臘字母指標的相互作用

希臘字母指標並非孤立運作。現實中的市場波動會同時改變多個因素，各希臘字母指標之間亦相互影響。

```
相互作用圖：

  +------------------+
  |   股票價格       |-----> 德爾塔（主要）
  |    變動          |-----> 伽馬（調整德爾塔）
  +------------------+       西塔（受價值狀態影響）
                             維加（受價值狀態影響）

  +------------------+
  |  時間流逝        |-----> 西塔（主要）
  |                  |-----> 伽馬（臨近到期時增大）
  +------------------+       維加（臨近到期時縮小）
                             德爾塔（平價期權維持~0.50）

  +------------------+
  |  波動率          |-----> 維加（主要）
  |   變動           |-----> 德爾塔（移向0.50）
  +------------------+       伽馬（高隱含波動率時下降）
                             西塔（高隱含波動率時上升）

  +------------------+
  |  利率            |-----> 羅（主要）
  |   變動           |       （與其他指標的相互作用較小）
  +------------------+
```

**關鍵相互作用一：伽馬與西塔相互對立**

這是最重要的相互作用。高伽馬的期權必然有高西塔。兩者不可兼得。這形成了一個根本性的取捨：

```
伽馬-西塔取捨：

  買入期權（長倉）：
    + 正伽馬（德爾塔朝有利方向移動）
    - 負西塔（每日為此支付代價）

  賣出期權（短倉）：
    + 正西塔（每日賺取時間值）
    - 負伽馬（德爾塔朝不利方向移動）

  你無法同時擁有正伽馬和正西塔。
  這是期權的鐵律。

  示例：買入平價認購期權
    德爾塔 = +0.50，伽馬 = +0.03，西塔 = -0.08，維加 = +0.10

  示例：賣出平價認購期權
    德爾塔 = -0.50，伽馬 = -0.03，西塔 = +0.08，維加 = -0.10

  每位期權交易者都面對的抉擇：
  「我願意支付西塔來持有伽馬，還是收取西塔並承受伽馬風險？」
```

**關鍵相互作用二：維加與時間**

距到期日較長的期權維加較大，但伽馬較小。距到期日較短的期權伽馬較大，但維加較小。這意味著隨著到期日臨近，主導風險因素會發生轉變。

```
按距到期日天數劃分的主導希臘字母指標：

  距到期日      主導指標        主要風險
  ===========   ============    =========================
  > 60天        維加            隱含波動率變動主導盈虧
  30-60天       維加/西塔       兩者均顯著影響
  14-30天       西塔/伽馬       消耗加速，伽馬上升
  < 14天        伽馬/西塔       伽馬風險主導
  < 3天         伽馬            平價期權面對純伽馬風險
```

**關鍵相互作用三：真實場景下的全部希臘字母指標**

```
完整希臘字母指標分析：

  倉位：買入5張AAPL $150認購期權，距到期日30天
  股票報價$148，隱含波動率 = 28%

  每張合約        x5張合約        含義
  ============    ============    ================================
  德爾塔: +0.45   +225            ~等值於持有225股多倉
  伽馬: +0.025    +12.5           每升$1，德爾塔增加12.5
  西塔: -$0.07    -$35/天         倉位每天持倉成本$35
  維加: +$0.10    +$50/每1%隱波   每1%隱含波動率上升，增值$50
  羅:   +$0.06    +$30/每1%利率   影響微乎其微

  情景A：股票升至$153（+$5），隱含波動率不變，5天過去
    德爾塔盈虧：  +$5 x 225 = +$1,125（約值，德爾塔會變動）
    西塔盈虧：    -$35 x 5 = -$175
    伽馬調整：    德爾塔從225升至~288，盈利加速
    淨值：        約 +$1,050

  情景B：股票維持$148，隱含波動率跌3%，5天過去
    德爾塔盈虧：  $0
    維加盈虧：    -3 x $50 = -$150
    西塔盈虧：    -$35 x 5 = -$175
    淨值：        約 -$325

  情景C：股票跌至$143（-$5），隱含波動率急升+5%，5天過去
    德爾塔盈虧：  -$5 x 225 = -$1,125（約值）
    維加盈虧：    +5 x $50 = +$250
    西塔盈虧：    -$35 x 5 = -$175
    伽馬調整：    德爾塔從225降至~163，虧損減速
    淨值：        約 -$900（維加部分抵消德爾塔虧損）
```

### c) 常見誤解

**誤解一：「德爾塔就是期權在到期時處於價內的概率。」**

德爾塔近似於這個概率，但並不等於它。期權定價模型中到期實值的數學概率，在布萊克-舒爾斯公式中以N(d2)表示，而德爾塔則是N(d1)。實際操作中，德爾塔用作快速估算已足夠接近，但若需要精確概率，應使用你的證券行平台的概率計算器。兩者差距在高波動率、長存續期期權中最為顯著。

**誤解二：「西塔是固定且可預測的。」**

西塔既非固定，也非完全可預測。它每天都會根據價值狀態、距到期日天數和隱含波動率而改變。一個期權今天的西塔可能是-$0.05，下週可能是-$0.08。切勿以當前西塔值作線性推算。此外，西塔消耗在交易日內並非均勻發生。學術模型假設連續消耗，但實際上，每日西塔的損失大部分在隔夜收市時計入。

**誤解三：「我只需關心德爾塔，因為它是最重要的希臘字母指標。」**

對於短期方向性交易而言，德爾塔確實是主導因素。但對於收入策略（如備兌認購期權和現金擔保認沽期權），西塔才是主要盈利驅動力。在業績交易中，維加的重要性可能超過德爾塔。「最重要」的希臘字母指標，完全取決於你的策略和時間維度。專業交易者會同時管理全部五個希臘字母指標。

**誤解四：「希臘字母指標是精確的預測。」**

希臘字母指標是基於當前條件和數學模型的瞬間衡量值。它們時刻在變。一個現時德爾塔為0.50的期權，在股票移動2美元後可能德爾塔變為0.55，即使原始德爾塔預測2美元股票移動的盈利為1.00美元（0.50 × 2）。這正是伽馬的作用。希臘字母指標是一階近似值，在短時間內的小幅變動中效果最佳。

**誤解五：「賣出期權因為有正西塔，等同於免費的錢。」**

正西塔是以承受負伽馬換取的。你每天收取少量收入，但須承受大幅波動帶來巨大虧損的風險。這份收入並非免費；它是承擔風險的補償。只考慮西塔而忽略伽馬的期權賣方，就像只計算保費而不考慮理賠的保險公司。遲早，一場巨大損失將會到來。

**誤解六：「備兌認購期權和現金擔保認沽期權不受維加影響。」**

維加的影響確實存在。若你在隱含波動率低時賣出備兌認購期權，之後隱含波動率急升，你的短倉認購期權升值，造成未實現虧損。你應在隱含波動率高時賣出期權（收取更多期權金，並受益於隱含波動率下跌），在隱含波動率低時買入期權。忽視維加意味著你可能長期賣出便宜期權，買入昂貴期權。

**誤解七：「羅無關痛癢，可以完全忽略。」**

對於短期期權，羅確實微不足道。但對於長期期權（存續期一年或以上），羅的影響不容小覷。在2022至2023年聯儲局快速加息周期中，羅的效應明顯影響了長期期權的價格。若你交易長期期權（我們將在第38週介紹），便需要將羅納入考量。

### d) 問答環節

**問：如何查看我的期權倉位的希臘字母指標？**

答：每個主要證券行平台都顯示希臘字母指標。在大多數平台中，你可以在期權鏈顯示中添加德爾塔、伽馬、西塔、維加和羅的欄位。當你持有倉位時，投資組合視圖通常顯示倉位層面的希臘字母指標（即每張合約的數值乘以合約數量）。若你的平台沒有顯著顯示希臘字母指標，可查找「期權分析」或「風險概況」部分。

**問：週末期間希臘字母指標會有變化嗎？**

答：希臘字母指標本身（敏感性衡量值）在週一開市後重新計算。然而，西塔消耗在週末同樣發生。大多數期權定價模型假設按365天計算消耗，即期權在兩天週末內損失約2/365的時間值。部分交易者對週末時間值的確切定價方式存有爭議，但從實際觀察來看，週五賣出期權、週一買回的操作通常能捕捉週末西塔。這對長線策略影響甚微，但對短期交易者有一定意義。

**問：如何計算整個投資組合的希臘字母指標？**

答：投資組合的希臘字母指標可直接相加。只需將每個倉位的德爾塔、伽馬、西塔和維加加總即可。多倉以其自然符號計入，空倉則取相反符號。大多數證券行平台會自動計算投資組合希臘字母指標。這正是專業交易者管理風險的方式：監控投資組合層面的希臘字母指標，當任一指標在某一方向過度集中時進行調整。

**問：什麼是「德爾塔中性」？我是否應該追求德爾塔中性？**

答：德爾塔中性倉位的淨德爾塔接近零，即小幅股票價格波動對倉位盈虧影響甚微。做市商持續維持德爾塔中性的帳簿。對於運用收入策略的散戶投資者，並不需要追求完美的德爾塔中性。然而，了解自己離中性的距離有助於風險管理。若你的投資組合德爾塔為+800，相當於持有800股等值的風險敞口。這可能合適，也可能過於進取，取決於你的風險承受能力和投資組合規模。

**問：我常在新聞中聽到「伽馬軋空」，這是什麼意思？**

答：當賣出認購期權的做市商需要買入股票對沖其倉位，且股票不斷上漲時，便會出現伽馬軋空。連鎖反應如下：散戶買入認購期權，做市商賣出這些認購期權，因此做市商持有短伽馬和短德爾塔倉位。為對沖，做市商按其德爾塔風險敞口買入股份。隨著股票上漲，認購期權的德爾塔增加（伽馬效應），做市商必須買入更多股份。這些額外的買盤進一步推高股價，德爾塔再度增加，形成正反饋循環。2021年1月GameStop（GME）事件，部分原因正是由伽馬軋空所驅動。

**問：持有買入期權倉位，每天的成本是多少？**

答：查看西塔數值。若西塔為每股-$0.05，即每張合約每日成本5美元。持有30天即每張合約150美元。將此成本與你預期從德爾塔（方向性波動）獲得的盈利，以及任何維加受益進行比較。若你在預計持倉期間，從股票波動中獲得的預期盈利，未能舒適地超越西塔成本，則從希臘字母指標角度來看，這筆交易的風險回報欠佳。

**問：當某個希臘字母指標變得極端時，我是否應該調整倉位？**

答：是的，但切勿過度調整。最常見的調整觸發因素有：（1）德爾塔相對於你的風險承受能力過大，提示你應對沖或縮減倉位；（2）臨近到期時短倉期權的伽馬急劇上升，提示你應平倉或展期；（3）在業績等已知波動事件前，維加敞口過大。對於收入策略（備兌認購期權、現金擔保認沽期權），最實際的調整方式是展期：平掉當前倉位，在不同行使價或到期日重建新倉位。我們在第27和28週已詳細討論了展期操作。

**問：布萊克-舒爾斯模型與真實市場的希臘字母指標一樣嗎？**

答：概念含義相同，但實際數值因模型而異。布萊克-舒爾斯假設波動率恆定，價格呈對數正態分佈。現實中，波動率不斷變化，價格分佈具有厚尾效應，且波動率微笑（不同行使價對應不同隱含波動率）也影響希臘字母指標。做市商使用更複雜的模型，以考慮這些因素。對於使用證券行平台希臘字母指標數值的散戶投資者而言，這些數值對於倉位管理已足夠準確。你無需深究數學模型之間的差異。
