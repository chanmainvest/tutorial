# 第26周：期权作为条件订单

## 阅读部分

### a) 为什么这很重要

大多数投资者认为期权是复杂、另类的工具，专供专业人士和投机者使用。这种思维定式形成了一道障碍，使他们无法运用一些简单的期权策略来大幅改善投资结果。本课将介绍一种能够改变一切的思维模型：**期权作为条件订单**。

核心洞见如下：当你卖出一份看跌期权时，你实质上是在为一只股票挂出一个限价买入订单，并在等待期间获得报酬。当你对已持有的股票卖出看涨期权时，你实质上是在挂出一个限价卖出订单，并在等待期间获得报酬。

这不仅仅是一个便于理解的类比，其经济结果几乎完全相同。区别在于：使用期权时，你会因为同意以目标价格买入或卖出而收取收益（期权费）；而使用普通限价单时，你只能守着现金干等，毫无收益。

**这对你的投资有何意义？**

**闲置现金是一种浪费。** 许多投资者将现金放在场外，等待合适的时机入场。也许他们在等苹果跌到140美元，或等市场回调10%。与此同时，这些现金只能赚取约4-5%的存款利率。如果改为在目标买入价卖出现金担保看跌期权，他们可以在等待同一入场点的同时，赚取8-20%的年化收益。若股价跌至目标价，他们就像执行限价单一样买入股票；若未跌到目标价，则保留期权费，下个周期再操作。

**卖出决策在情感上很困难。** 许多投资者因难以下定决心而持有赢利股票过久。通过在目标卖出价卖出备兑看涨期权，他们可以提前承诺执行纪律严明的退出计划，并获得相应报酬。期权费收入在一定程度上也缓解了卖出盈利仓位时的情感压力。

**这种方法与长期持有理念高度契合。** 你并非在投机短期走势，而是利用期权实践每本投资教材都推崇的低买高卖纪律，同时在此过程中获取收益。这是价值投资者使用期权作为工具的方式，而非日内交易者的投机手段。

一旦你将这一思维模型内化，你将永远不会以同样的眼光看待现金担保看跌期权或备兑看涨期权。它们将成为你作为纪律投资者已有行为的自然延伸。

### b) 你需要了解的内容

#### 限价单类比

在讨论期权之前，我们先来回顾限价单的运作方式。

**限价买入单**告诉你的券商："帮我买这只股票，但只能以这个价格或更低的价格成交。"如果股价始终未达到你的限价，该订单永远不会成交，你的现金也不会动用。

**限价卖出单**告诉你的券商："帮我卖出这只股票，但只能以这个价格或更高的价格成交。"如果股价始终未达到你的目标价，你就继续持有股票。

```
传统限价单：

限价买入单：                          限价卖出单：
"以140美元或更低价格买入AAPL"         "以180美元或更高价格卖出AAPL"

股价155美元：订单未成交                股价155美元：订单未成交
股价跌至140美元：订单成交              股价涨至180美元：订单成交
             以140美元买入                          以180美元卖出

等待期间：现金收益为零                等待期间：无额外收入
         （或极少利息）                           （仅有股息）
```

现在让我们看看卖出期权如何与此完全对应，同时还能带来额外收益。

#### 将卖出看跌期权视为限价买入单

当你卖出（卖开）一份看跌期权时，你做出了一项承诺："如果在到期日前该股票跌至或低于行权价，我同意以该行权价买入100股。作为对这项承诺的回报，我预先收取期权费。"

这在经济上等同于一份限价买入单，并附带一项重大提升：**你在等待期间获得了报酬**。

```
对比：限价买入单 vs. 卖出看跌期权

+------------------------------------------------------------------+
|                  限价买入单            |    卖出看跌期权           |
+------------------------------------------------------------------+
| 操作：          "以140美元买入"        | "以3.00美元卖出140行权价   |
|                                       |  看跌期权"               |
| 备用资金：       14,000美元            | 14,000美元（现金担保）    |
| 若股价跌至                            |                          |
|   140美元：     以140.00美元买入      | 以140美元买入，保留期权费 |
|                 成本：140美元/股       | 实际成本：137美元/股      |
| 若股价保持在                          |                          |
|   140美元以上：  无任何操作           | 保留300美元期权费         |
|                  现金收益约为零        | 约30天内赚取300美元(2.1%) |
| 时间限制：       长期有效              | 在特定日期到期            |
| 等待期间                              |                          |
|   的收入：       0美元                | 每张合约300美元           |
+------------------------------------------------------------------+
```

让我们通过一个完整的示例来详细说明。

**场景：** 你想买入苹果（AAPL），目前股价155美元，但你认为140美元是合理的买入价。你手头有14,000美元现金。

**传统方式：** 挂出100股的限价买入单，限价140美元。等待。如果苹果从未跌至140美元，你的14,000美元就闲置在那里，几乎毫无收益。

**期权方式：** 卖出1张AAPL行权价140美元、30天后到期的看跌期权，收取期权费1.85美元/股（每张合约185美元）。

```
结果分析：

情景一：AAPL收于140美元以上（最可能发生）
  - 你的看跌期权到期作废
  - 你保留185美元期权费
  - 收益：185美元 / 14,000美元 = 30天内1.32% = 约16.1%年化
  - 你仍持有14,000美元现金，下个月可再次卖出看跌期权

情景二：AAPL跌至140美元（你的目标价）
  - 你被行权：以140美元买入100股
  - 实际买入成本：140美元 - 1.85美元 = 138.15美元/股
  - 这比限价单的成交价还要优惠
  - 你以更低的折扣价买入了你想要的股票

情景三：AAPL跌至130美元（低于行权价）
  - 你被行权：以140美元买入100股
  - 股票现价130美元，账面亏损10美元/股
  - 但你的实际成本为138.15美元，而你本来就想买这只股票
  - 结果与限价单在140美元成交后股价继续下跌相同
  - 实际上更优，因为你的成本基础是138.15美元，而非140美元

情景四：AAPL短暂跌至140美元后反弹至160美元
  - 你可能被行权，也可能未被行权（取决于时机）
  - 若被行权：实际成本138.15美元买入，股价现为160美元 = 盈利
  - 若未被行权：保留期权费，再次卖出看跌期权
```

请注意一个重要事实：**在任何情景下，看跌期权卖方的结果都不会差于限价单买方。** 在每种情况下，期权卖方要么获得相同结果加上期权费收入，要么获得更好的买入价格。唯一的权衡是时间限制（期权有到期日，而限价单可以无限期挂单）。

#### 期权费：获得等待报酬

你在卖出看跌期权时收取的期权费不仅仅是一个额外奖励，它是对以下三件事的补偿：

1. **时间承诺：** 你将资金锁定了一段特定时间。
2. **风险承担：** 即使股价大幅下跌，你也承担了买入的义务。
3. **机会成本：** 当你的现金为这笔潜在买入而保留时，你无法将其用于其他地方。

```
决定期权费大小的因素：

+---------------------------------------------------------------+
| 因素                        | 对期权费的影响                |
+---------------------------------------------------------------+
| 隐含波动率越高              | 期权费越大（风险越高）        |
| 距到期日时间越长            | 期权费越大（时间越多）        |
| 行权价越接近当前股价        | 期权费越大（被行权可能性越高）|
| 行权价距当前股价越远        | 期权费越小（被行权可能性越低）|
| 到期日前有财报发布          | 期权费越大（事件风险）        |
+---------------------------------------------------------------+

期权费示例（AAPL股价155美元，距到期日30天）：

  行权价155美元（平值）：期权费约4.20美元（30天内约2.7%）
  行权价150美元（虚值3%）：期权费约2.50美元（30天内约1.7%）
  行权价145美元（虚值6%）：期权费约1.40美元（30天内约1.0%）
  行权价140美元（虚值10%）：期权费约0.70美元（30天内约0.5%）
  行权价135美元（虚值13%）：期权费约0.30美元（30天内约0.2%）

权衡关系：
  越接近平值 = 期权费越高，但被行权的概率也越高
  越深度虚值 = 期权费越低，但被行权的概率也越低
```

对于大多数将此策略作为"有偿限价单"的保守型投资者来说，虚值5-10%的行权价提供了较好的平衡：合理的期权费、较低的被行权概率，以及一个你真正满意的买入价格。

#### 将卖出看涨期权视为限价卖出单

同样的思维模型反向适用于备兑看涨期权。当你对已持有的股票卖出看涨期权时，你的意思是："如果在到期日前股票涨至行权价以上，我同意以该行权价卖出我的100股。作为回报，我预先收取期权费。"

这在经济上等同于一份限价卖出单，并在等待期间带来收益。

```
对比：限价卖出单 vs. 卖出看涨期权（备兑）

+------------------------------------------------------------------+
|                  限价卖出单            |    备兑看涨期权           |
+------------------------------------------------------------------+
| 操作：          "以180美元卖出"        | "以2.00美元卖出180行权价  |
|                                       |  看涨期权"               |
| 持有股数：       100股                 | 100股                    |
| 若股价涨至                            |                          |
|   180美元：     以180.00美元卖出      | 以180美元卖出，保留期权费 |
|                 每股所得：180美元      | 实际价格：182美元/股      |
| 若股价保持在                          |                          |
|   180美元以下：  无任何操作           | 保留200美元期权费         |
|                  无额外收入            | 赚取200美元（收益）       |
| 时间限制：       长期有效              | 在特定日期到期            |
| 等待期间                              |                          |
|   的收入：       0美元（仅股息）      | 200美元 + 股息            |
+------------------------------------------------------------------+
```

**完整示例：**

你以155美元买入了100股苹果，若能以180美元卖出（16%涨幅）则感到满意。你没有挂出限价卖出单，而是卖出了一张备兑看涨期权。

```
操作设置：
  持有：100股AAPL，成本基础155美元
  卖出：1张AAPL 180美元看涨期权，45天后到期，期权费1.50美元（每张合约150美元）

情景一：AAPL收于180美元以下（最可能发生）
  - 看涨期权到期作废
  - 你保留100股股票和150美元期权费
  - 期权费收益率：150美元 / 15,500美元 = 45天内0.97% = 约7.9%年化
  - 你可以下个月再次卖出看涨期权
  - 全年潜在总收入：股息 + 期权费 = 约8-10%

情景二：AAPL涨至180美元以上
  - 股票被行权（卖出），成交价180美元
  - 总收入：180美元 + 1.50美元期权费 = 每股181.50美元
  - 盈利：181.50美元 - 155美元 = 每股26.50美元 = 17.1%收益
  - 这比在180美元的限价卖出单结果更好
  - 是的，你错过了181.50美元以上的涨幅，但你以目标价卖出了股票

情景三：AAPL跌至140美元
  - 看涨期权到期作废，你保留150美元期权费
  - 你仍持有股票（此时股票仓位亏损）
  - 150美元期权费略微缓解了亏损
  - 成本基础降低：155美元 - 1.50美元 = 153.50美元
```

#### 风险特征对比：限价单 vs. 卖出看跌期权

尽管这一类比很有说服力，理解两者在风险特征上的差异同样重要。

```
风险对比：以140美元限价买入 vs. 卖出140美元看跌期权

                     限价买入单              卖出看跌期权（期权费3美元）
                     ================        =====================
最大盈利：           无限（股价上涨）        300美元（仅限期权费）
                                            （若未被行权）

盈亏平衡点：         140.00美元             137.00美元（140 - 3）

最大亏损：           14,000美元（股价归零）  13,700美元（股价归零）

所需资金：           14,000美元（成交时）    14,000美元（现在即锁定）

等待期间收入：       0美元                  每周期300美元

行权                 精确价格              近似价格
精确度：             （恰好在140美元成交）  （在140美元被行权）

部分成交：           可买入更少股数          必须精确买入100股
                     （例如37股）           每张合约

时间期限：           无限期（长效有效）      固定到期日

现金灵活性：         撤单后资金释放          到期日前锁定
                                           （或提前平仓）
```

需要注意的关键差异：

1. **精确性与收益的权衡：** 限价单会以你精确设定的价格成交，而看跌期权为你带来收入，但要求以100股为单位，在固定行权价买入。

2. **部分成交：** 限价单可以部分成交（如买入37股），而期权只能以100股为单位。如果你想买入250股，可以卖出2张看跌期权合约（覆盖200股），剩余50股则用限价单处理。

3. **时间限制：** 限价单可以无限期挂单（长效有效）。期权有到期日。你需要在每次到期后卖出新的看跌期权，主动管理该策略。

4. **跳空风险：** 若股票因坏消息隔夜大幅跳空下跌（例如从155美元跌至120美元），限价单和看跌期权都会导致以亏损价格买入股票。然而，限价单买方可能在140美元成交后继续看着股价下跌，而看跌期权卖方虽在140美元被行权，但有期权费的缓冲垫（实际成本137美元）。

#### 何时使用此方法

该策略在特定情况下最为有效：

```
卖出看跌期权（有偿限价买入）的理想条件：

  [是]  你真心想在行权价买入该股票
  [是]  你有足够的资金买入100股
  [是]  你有一个低于当前市价的目标买入价
  [是]  该股票具有合理的隐含波动率（期权费可观）
  [是]  若被行权，你愿意长期持有该股票
  [是]  该公司基本面扎实

  [否]  你试图投机短期走势
  [否]  若被行权，你无力买入该股票
  [否]  你在任何价格下都不想持有该股票
  [否]  该股票极度波动或面临财务困境
  [否]  隐含波动率异常偏低（期权费微乎其微）
```

```
卖出看涨期权（有偿限价卖出）的理想条件：

  [是]  你持有至少100股该股票
  [是]  你有一个高于当前市价的目标卖出价
  [是]  你乐意在行权价卖出
  [是]  该股票处于横盘或温和上涨趋势中
  [是]  你希望在股息之外获得额外收益

  [否]  你对该股票极度看多，预期将大幅上涨
```

### c) 常见误解

**误解一："卖出看跌期权与做空股票相同。"**

这是一种危险的混淆。做空股票是指借入股票并卖出，希望以更低价格买回。你的风险理论上是无限的，因为股价可以无限上涨。而卖出看跌期权意味着你同意以特定价格**买入**股票。你的最大亏损仅限于行权价减去期权费（若股价归零）。这是两种截然不同、风险特征完全不同的策略。做空是看空操作，而卖出看跌期权是中性偏多操作。

**误解二："你应该对你不想持有的股票卖出看跌期权，因为你永远不会被行权。"**

这种想法极其危险。虽然大多数虚值看跌期权确实到期作废，但真正被行权的那些可能造成巨大损失。如果你对一只你不想持有的股票卖出看跌期权，而它下跌40%，你将被迫以远高于当前市价的价格买入100股你毫无信心的公司股票。只对你真正愿意在行权价买入的股票卖出看跌期权。

**误解三："卖出备兑看涨期权意味着你会错过下一次大涨行情。"**

备兑看涨期权确实会限制你的上行收益，但上限是你**自己选择**的目标卖出价格。如果AAPL股价155美元，你卖出180美元看涨期权，你的意思是愿意以180美元卖出（16%涨幅）。若AAPL大涨至220美元，是的，你"错过"了40美元的额外涨幅。但你仍然获得了16%的利润加上期权费。错过意外之财并不等同于亏损。而且从统计数据来看，绝大多数月份，股价会在行权价以下收盘，你只需保留期权费即可。

**误解四："期权费太小，不值得承担这种风险。"**

单次期权费看起来可能不大，但它们随时间复利增长。在15,000美元仓位上每月赚取200美元期权费，每年就是2,400美元，年化收益率16%。即使是保守的每月100美元，也是每年1,200美元，即8%的年化收益。这还是在股息和资本增值之外额外获得的。在20年投资生涯中，这些额外收入经再投资后，可为你的投资组合增加数十万美元的财富。

**误解五："卖出看跌期权的风险与买入股票完全相同。"**

虽然最大亏损相近（若公司破产，两者损失均接近全部），但看跌期权卖方实际上拥有更优的买入价格。收到的期权费降低了实际成本基础。若你以行权价140美元卖出期权费3美元的看跌期权并被行权，你的实际成本是137美元；而直接在140美元买入股票的投资者成本是140美元。在每种情景下，看跌期权卖方都比每股优惠3美元。两者的风险特征并不相同，期权卖方的结果实际上略优。

**误解六："卖出期权时你需要时刻盯盘。"**

对于本课所介绍的策略，你卖出期权后基本上只需等待。你可能每周查看一次，了解是否需要采取任何行动，大约需要5分钟。月度期权仅需在到期周前后多加关注。这不是日内交易，更像是设好陷阱定期查看的工作。许多投资者每周花15-30分钟管理他们的期权仓位。

### d) 常见问题与解答

**Q1：如果我卖出一张看跌期权，股票隔夜跳空下跌30%，会发生什么？**

A：你将在行权价被行权，而股票的实际价值将远低于你的买入价格。这是卖出看跌期权的主要风险。然而，这与使用限价买入单或直接买入股票面临的风险相同。如果你卖出AAPL 140美元看跌期权，而AAPL跌至100美元，你以140美元买入（含期权费后实际成本137美元），立刻面临每股37美元的账面亏损。这就是为什么你应该只对基本面强劲、你相信会复苏的股票卖出看跌期权。在这种情况下，3美元的期权费缓冲虽然相对于亏损来说很小，但仍然优于以140美元成交的限价单，毫无任何缓冲。

**Q2：如果股价开始下跌，我可以提前平仓吗？**

A：可以。你随时可以在到期前买回卖出的看跌期权。如果股价下跌，看跌期权价值上升，你需要支付高于当初收取的价格，产生净亏损。但这使你能够管理风险。例如，一个常用原则是：当亏损超过期权费收入的2倍时平仓。如果你以3.00美元卖出看跌期权，现在报价为9.00美元，你可以以6.00美元的亏损买回，而非承担被行权的风险。反过来，若股价保持平稳或上涨，期权价值衰减，你可以提前以盈利平仓。一个常用原则是：当已获取最大期权费的50-75%时买回平仓。

**Q3：为什么不直接用限价单？为什么要用期权把事情复杂化？**

A：对于小额仓位（少于100股）或需要精确价格成交的情况，限价单完全没问题。在以下情况下，期权能创造额外价值：（1）你希望在等待目标价期间赚取收入；（2）你本来就会以100股为单位买入；（3）你希望随时间系统性地建仓；（4）你希望你的现金更努力地为你工作。一旦你理解了操作机制，所谓的"复杂性"便微乎其微。在大多数券商平台上，卖出一张现金担保看跌期权大约只需2分钟。

**Q4：如果卖出看跌期权后我改变主意，不想买入该股票了，怎么办？**

A：买回看跌期权来平仓。如果时间已过，且股价未下跌，看跌期权的价格很可能低于你卖出时的价格，你将获利了结。如果股价下跌，看跌期权价格更高，你将承受小额亏损。随时平仓的能力是交易所上市期权的优势之一。你永远不会被真正"锁死"在仓位中。

**Q5：期权费的税务处理如何？**

A：如果看跌期权到期作废，期权费属于短期资本利得，按你的普通所得税率征税。如果被行权，期权费将降低你所购股票的成本基础，影响你未来卖出股票时的资本利得计算。对于备兑看涨期权，若看涨期权到期作废，期权费属于短期资本利得；若被行权并卖出股票，期权费将计入你的卖出所得。具体情况请咨询税务专业人士。

**Q6：我可以在退休账户（IRA）中卖出看跌期权吗？**

A：可以，大多数券商允许在IRA账户中进行现金担保的看跌期权操作。由于不能使用保证金，必须备有全额现金（因此称为"现金担保"）。IRA账户中同样可以进行备兑看涨期权操作。IRA账户的税务优势意味着期权费可以享受税收递延增长（传统IRA）或免税增长（Roth IRA），使这些策略在退休账户中更具吸引力。

**Q7：如何选择不同的到期日？**

A：对大多数投资者而言，最佳选择是距到期日30-45天。较短期限（周度）每次交易的期权费较低，且需要更频繁的主动管理。较长期限（60-90天）提供更多期权费，但资金占用时间更长，Theta衰减的速度也不那么理想。30-45天的范围提供了期权费收入与管理精力之间的最佳比率。如果你偏好减少交易频率，45-60天也很适合。如果你希望最大化年化收益率，21-30天最为理想。

**Q8：如果我想买入150股怎么办？期权只能以100股为单位。**

A：采用组合方式。卖出1张看跌期权合约（覆盖100股），再为剩余50股挂出限价单。或者，如果你有耐心，先卖出1张看跌期权合约，被行权后再卖出第二张看跌期权合约买入另外100股，之后再单独处理剩余50股。期权最适合100股的整数倍仓位。

**Q9：如果公司宣布并购，我的看跌期权会怎样？**

A：若公司被收购，期权将根据交易条款进行调整。若为现金收购，期权可能以现金结算；若为换股收购，期权合约将被调整为交割新股。这些调整由期权结算公司（OCC）处理。虽然具体操作机制可能较为复杂，但作为期权持有方，你的权利不会受损，作为期权卖方，你的义务也不会超出原始合约约定的范围。

**Q10：操作是否有最优顺序？是先卖看跌期权，再卖备兑看涨期权吗？**

A：一个自然的操作流程是：（1）确定你想持有的股票；（2）在目标买入价卖出看跌期权，在赚取收益的同时逐步建仓；（3）持有股票后，在目标卖出价卖出备兑看涨期权，赚取额外收益。这个循环——卖出看跌期权、被行权买入、卖出备兑看涨期权、被行权卖出、再次卖出看跌期权——被称为"滚轮策略"，我们将在第28周对此进行深入讲解。
