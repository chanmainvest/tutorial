# Review: course/level2_overview.md

## Overall Assessment

The Level 2 overview is concise and gives a clean map of Weeks 13-24. It correctly signals a shift from foundational portfolio construction into active strategies, performance analysis, corporate fundamentals, valuation, factors, and multi-strategy construction.

The overview is too generic for the course's distinctive philosophy. It should make clear that Level 2 is not a permission slip to become a retail trader. It is an apprenticeship in understanding tools, risks, and when not to use them. It also needs structural and terminology cleanup.

## Content Critique

- The weekly outline is useful and mostly matches the visible course map.
- The phrase "thinking like a serious investor" is good, but should be defined as better process, risk control, and evidence discipline, not more activity.
- The short-selling introduction needs a risk caveat: unlimited loss, borrow cost, squeeze risk, and solvency risk.
- Pair trading should be framed as a lesson in relative value and risk controls, not easy arbitrage.
- "Take your portfolio global" conflicts with the course's US-first investable-universe view unless the orthodox global case is taught first and then Horace's caveat is introduced.
- "A portfolio that a professional would respect" is a little vague. Better: a portfolio whose risks, benchmark, exposures, and rationale can be explained clearly.

## Cross-Reference And Consistency Issues

- The file does not use the required decimal H3 section numbering.
- The YouTube host description calls Horace an "experienced retail trader," which violates the repository guidance to use retail investor, not trader.
- The overview should reconcile Level 2's international diversification and multi-asset framing with SOUL principle #16.

## Presentation Improvements

- Add a "Level 2 is about tools, not more trades" section.
- Add a warning box for shorting, leverage, and active-strategy complexity.
- Add a progression chart: L1 core -> L2 tools -> L3 options -> L4 portfolio/risk -> L5 advanced integration.
- Add a table with each week's practical output: hedge map, pair-trade template, rate-duration checklist, valuation worksheet, factor exposure map.
- Add a prerequisite checklist: Level 1 portfolio built, IPS written, emergency cash, no high-interest debt, understands drawdown risk.

## YouTube Script Critique

The script is simple and serviceable, but not yet compelling. It reads like a syllabus promo rather than a retention-oriented video.

Specific improvements:

- Start with Stella asking: "I have my first portfolio. Why shouldn't I just stop here?"
- Horace should answer: "You can stop, but Level 2 teaches what can break it, how to evaluate alternatives, and how to avoid being sold complexity."
- Replace "retail trader" with "retail investor."
- Add one tension point: shorting can protect a portfolio or destroy one.
- End with a clear promise: by the end of Level 2, students can explain their portfolio's exposures and evaluate whether a strategy is worth its complexity.

## Chart And Visual Feedback

No charts are necessary, but a Level 2 roadmap visual would improve orientation.

Recommended visual:

- Four blocks: active strategy, macro/asset allocation, company analysis, valuation/factors/integration.
- Each block connects back to the Level 1 portfolio base.

## Interactive Demo Feedback

No interactive is required for this overview. A clickable roadmap that jumps to each week would be useful on the website.

## New Interactive Demo Ideas

- Level 2 readiness checklist.
- Strategy complexity ladder: passive core, tilts, hedges, shorts, pair trades, multi-strategy.
- Portfolio exposure preview showing what Level 2 adds to the Week 12 first portfolio.

## Make It More Entertaining Without Watering It Down

Make Level 2 feel like learning the controls in the cockpit, not being encouraged to press every button. The tension is: more tools can improve your portfolio, but more tools can also hurt you if you mistake complexity for edge.

## Money-Making Usefulness

This overview can help students make money by setting the right expectation: Level 2 is about better decisions and avoiding bad active strategies, not chasing action.

## SOUL Consistency Flags

- Needs stronger alignment with SOUL's skepticism toward easy alpha and its emphasis on solvency.
- Must replace "retail trader" language.
- Should state that orthodox concepts are taught first, then Horace's retail-investor caveats.
