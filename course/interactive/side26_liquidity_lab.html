Effective spread = quoted spread x volatility multiplier. Market impact uses an Almgren-Chriss square-root model: bps ~ 10 x sqrt(size / 1%-ADV) x vol-multiplier. Round-trip cost = spread + 2 x impact (one in, one out). Liq-adjusted return = gross return - trips/yr x round-trip cost.
