# 第五級：專家（第47-52週）

---

## 第一部分：閱讀章節

---

### 級別概覽

第五級是課程的總結。在六週內，您將掌握最進階的投資組合策略，並將所有知識整合成一套完整的專業級投資框架。這些是捐贈基金、家族辦公室和頂尖投資組合管理人所使用的技術。

### 您將學到什麼

**第47-49週：進階策略**
- 尾部風險避險——保護投資組合免受黑天鵝事件衝擊
- 結構型商品與明確結果型投資
- 波動率套利與利用變異數風險溢酬

**第50-52週：投資組合整合**
- 因子傾斜與另類風險溢酬——收割多種系統性報酬來源
- 管理期貨與趨勢追蹤——在其他資產全面崩跌時仍能獲利的危機阿爾法
- 完整投資組合整合——建構全天候投資組合

### 完成第五級後您將能夠做到

- 以合理成本為投資組合對沖極端市場事件
- 評估結構型商品並理解其內嵌選擇權
- 利用波動率期間結構獲取額外報酬
- 建構針對特定風險溢酬的因子傾斜投資組合
- 了解趨勢追蹤策略及其作為危機阿爾法的角色
- 建立一個能在不同市場環境下表現穩健的全天候投資組合

### 每週課程

| 週次 | 主題 |
|------|-------|
| 47 | 尾部風險避險 |
| 48 | 結構型商品與明確結果型投資 |
| 49 | 波動率套利與期間結構 |
| 50 | 因子傾斜與另類風險溢酬 |
| 51 | 管理期貨與趨勢追蹤 |
| 52 | 投資組合整合——全面匯總 |
