# Review: course/level2_reflection.md

## Overall Assessment

The Level 2 reflection is a useful behavioral checkpoint. It correctly warns that the danger after learning active tools is not ignorance but undisciplined activity.

It needs formatting and philosophy alignment work. The section numbering does not follow the course convention, and the non-US allocation prompts conflict with the course's US-only/SOUL house view unless framed as orthodox diversification rather than direct instruction.

## Content Critique

- The core message is strong: Level 2 creates decision-makers, and decision-making needs process.
- `Boredom is not an investment thesis` is a keeper line.
- The reflection should ask for a decision journal, not only a thesis sentence. Level 2 students need timestamped thesis, expected holding period, risk limit, and review date.
- The prompt to compute Sharpe, volatility, and max drawdown is good, but should warn that one year of data is noisy.
- The prompt to add a non-US asset class conflicts with the course's later house view that US-only is the default investable universe. Reframe as: understand orthodox diversification, then decide whether your IPS permits it.
- The example commitment `move 15% into international equities` also conflicts with SOUL unless marked as an example for students choosing an orthodox global-diversification path.
- The sector-cycle prompt is useful but should warn against overconfident cycle timing, which Week 10 already flagged.
- The reflection should explicitly say: do not add options in Level 3 until your Level 2 active decisions are documented and measurable.

## Structure And Formatting Issues

- Top-level reading sections are not numbered. They should use the course convention: `### 1. Why This Reflection Matters`, `### 2. Pause and Ask Yourself`, etc.
- There are no `#### 2.x` subsections, which is acceptable for a short reflection, but top-level numbering should still be fixed.
- The file has no static visual. A reflection worksheet image or decision-journal template would make it more concrete.

## YouTube Script Critique

The script is concise and watchable. The airplane/instrument-panel analogy works.

Specific improvements:

- Add a quick Stella example of a trade she made for a weak reason; it will make the reflection less abstract.
- Add a visible decision-journal template as the main visual.
- Replace `add one non-US asset class` with a more SOUL-consistent prompt: decide whether global diversification belongs in your IPS and document why.
- The outro is strong and should stay.

## Chart And Visual Feedback

Recommended visual:

- One-page decision journal template: thesis, evidence, counterargument, sizing, stop/kill rule, review date, outcome, process grade.
- A small dashboard mockup: return, volatility, drawdown, Sharpe, factor tilt.

## SOUL Consistency Flags

- Needs orthodox-vs-Horace framing for non-US assets.
- Should use `retail investor`, not imply active trading as the default identity.
- Should align active decision-making with SOUL's expression-toolkit idea: choose tools only when they express a clear thesis with survivable sizing.
