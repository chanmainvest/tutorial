# 第五周：债券——你投资组合的减震器

## 阅读部分

### a) 为什么这一课至关重要

债券是初级投资者最容易忽视的资产类别，没有之一。股票占据着财经头条和饭局谈资，债券却在默默无闻地做着那份不讨好的工作——在市场崩溃时撑住整个投资组合。如果你想成为一名认真的投资者，了解债券不是可选项，而是必修课。

以下是债券对你投资之路至关重要的原因：

**投资组合稳定性（减震器效应）：** 2008年金融危机期间，标普500指数下跌约37%。同一时期，美国国债上涨了约20%。持有100%股票的投资者，眼看着自己的50万美元投资组合缩水至31.5万美元。而持有60%股票和40%债券的投资者，投资组合仅下跌至约39.5万美元，体验要好得多。两个投资组合之间8万美元的差距，就是债券减震器效应的体现。当你看着全世界陷入恐慌时，债券是你投资组合中那个能让你安心入睡的部分。

**可预期的收入：** 与可以随时削减或取消的股息不同，债券的票息支付是合同义务。当你购买债券时，发行方在法律上承诺按固定计划向你支付固定金额的利息。这种可预期性对于需要支付日常开支的退休人员极为宝贵，对于希望投资组合中有稳定收入来源的年轻投资者同样有用。

**本金保护：** 如果你持有债券至到期日，且发行方未发生违约，你可以全额收回本金。这与股票有根本区别——股票不保证你能收回初始投资。对于你承受不起损失的资金，或有明确时间节点的目标（五年后买房、十年后支付学费），债券能提供股票根本无法匹敌的确定性。

**读懂经济运行：** 全球债券市场规模大于股票市场。债券市场往往比股票市场更早发出经济变化的信号。收益率曲线、信用利差，以及利率与资产价格之间的关系，这些概念都植根于债券市场。如果你不懂债券，你就错过了市场提供的关键信息的一半。

**真正有效的分散化：** 真正的分散化，意味着持有行为模式各异的资产。债券，尤其是政府债券，历史上在市场承压期间与股票呈现低相关甚至负相关。这一点并非许多所谓"分散化工具"所能企及。当股票崩盘时，优质债券往往因投资者避险而上涨。这种在危机中的负相关性，是债券在投资组合中不可替代的根本原因。

本课将带你从零开始建立对债券的理解。学完之后，你将明白债券的运作机制、其行为模式背后的原因，以及如何将债券作为投资组合的稳定基石加以运用。无论你是25岁还是65岁，无论你有5,000美元还是50万美元，债券在你的理财规划中都有其位置。

全球债券市场规模约为130万亿美元，超过全球股票市场。每天，债券价格都在传递关于通胀预期、经济增长以及政府与企业信用状况的信号。忽视债券的投资者，相当于对市场提供的一半信息视而不见。学完本课之后，你将不再是其中之一。

### b) 你需要掌握的知识

#### 什么是债券？

债券本质上是一笔贷款。当你购买债券时，你是在把钱借给发行方（政府、企业或市政机构），以换取两项承诺：定期利息支付（称为票息）以及在未来某个指定日期（称为到期日）归还本金（称为面值或票面价值）。

这是股票与债券的根本区别。购买股票，你成为公司的部分所有者。购买债券，你成为公司的贷款人。所有者有上涨潜力，但没有任何保障。贷款人获得合同保障，但上行空间有限。

```
+------------------------------------------------------------------+
|                       债券合同                                    |
|                                                                  |
|  你（投资者 / 贷款方）                                            |
|    |                                                             |
|    |--- 出借1,000美元（面值 / 票面价值）------>  发行方            |
|    |                                            （借款方）        |
|    |<-- 每6个月收取票息支付 -------------------  |               |
|    |<-- 到期日收回1,000美元 -------------------  |               |
|                                                                  |
|  示例：10年期美国国债，票息率4%                                   |
|                                                                  |
|  面值：      1,000美元                                           |
|  票息率：    每年4%（每半年支付一次）                             |
|  支付金额：  每6个月20美元（每年40美元）                          |
|  到期日：    自发行日起10年                                       |
|  到期时：    你取回1,000美元                                     |
+------------------------------------------------------------------+
```

#### 债券的核心要素

每只债券都包含以下基本特征：

**面值（票面价值）：** 发行方承诺在到期日偿还的金额。大多数债券面值为1,000美元。当人们说债券"平价交易"时，意味着它以1,000美元（即面值的100%）交易。

**票息率：** 发行方每年支付的利率，以面值的百分比表示。1,000美元债券的5%票息意味着每年50美元。大多数债券每半年支付一次票息，因此你每六个月将收到25美元。

**到期日：** 发行方必须偿还面值的日期。债券按到期日分类如下：

```
+-------------------------------------------------------------------+
|  到期日分类                                                        |
+-------------------------------------------------------------------+
|  短期：    1-3年    （国库券、短期国债）                           |
|  中期：    3-10年   （中期国债、公司债券）                         |
|  长期：    10-30年  （长期国债、长期公司债）                       |
+-------------------------------------------------------------------+
|                                                                   |
|  时间轴：                                                         |
|                                                                   |
|  今天         3年          10年              30年                  |
|    |------------|-------------|-----------------|                 |
|    [短期]      [中期]          [    长期       ]                  |
|                                                                   |
|  风险较低 <---------------------------------> 风险较高             |
|  收益率较低 <-------------------------------> 收益率较高           |
|  价格敏感度较低 <---------------------------> 价格敏感度较高        |
+-------------------------------------------------------------------+
```

**收益率：** 这才是有趣的地方。收益率是你持有债券实际获得的回报，它不一定等于票息率。理解票息率与收益率的区别，是债券投资中最重要的概念之一。

#### 票息率 vs. 收益率：理解两者的区别

票息率在债券发行时确定，此后永不改变。但债券发行后，其价格可在二级市场上波动。价格变化，实际回报（收益率）也随之变化。

```
+-------------------------------------------------------------------+
|              票息率 vs. 当前收益率                                 |
+-------------------------------------------------------------------+
|                                                                   |
|  债券：面值1,000美元，票息率5% = 每年支付50美元                   |
|                                                                   |
|  情景一：债券以平价交易（1,000美元）                              |
|    当前收益率 = 50 / 1,000 = 5.0%                                |
|    收益率等于票息率                                               |
|                                                                   |
|  情景二：债券以折价交易（900美元）                                |
|    当前收益率 = 50 / 900 = 5.56%                                 |
|    收益率高于票息率                                               |
|    （你付出更少，但仍收到相同的50美元）                           |
|                                                                   |
|  情景三：债券以溢价交易（1,100美元）                              |
|    当前收益率 = 50 / 1,100 = 4.55%                               |
|    收益率低于票息率                                               |
|    （你付出更多，但仍只收到50美元）                               |
|                                                                   |
+-------------------------------------------------------------------+
|                                                                   |
|  核心要点：价格与收益率朝相反方向运动                             |
|                                                                   |
|  价格上涨   ------>  收益率下降                                   |
|  价格下跌   ------>  收益率上升                                   |
+-------------------------------------------------------------------+
```

**到期收益率（YTM）：** 当前收益率仅考虑票息支付与价格的关系。到期收益率更为全面，因为它还考虑了债券到期以面值赎回时所产生的盈亏。

```
到期收益率的计算涵盖：
  1. 每年的票息支付
  2. 购买价格与面值之间的差额
  3. 距到期日剩余的时间
  4. 再投资票息的复利效应

示例：
  以950美元购入债券，票息率5%，距到期日5年

  当前收益率：50 / 950 = 5.26%

  到期收益率还包括：
    到期时的资本利得：1,000 - 950 = 50美元
    分摊至5年：每年额外收益10美元
    到期收益率 = 约5.8%

  到期收益率 > 当前收益率（因为折价买入也带来额外收益）
```

到期收益率是比较债券的最佳单一指标，因为它涵盖了来自票息支付、价格升贬值及货币时间价值的全部预期回报。

#### 反向关系：价格与收益率

这可以说是整个债券投资领域最重要的概念。债券价格与收益率始终朝相反方向运动，没有例外。理解其背后的原因至关重要。

想象你持有一只票息率为3%的债券。此时利率上升，新发行的债券票息率为5%。当新发行的5%债券唾手可得，谁会愿意以全价买入你那只3%的旧债券？没有人。因此，你的债券价格必须下跌，直到其收益率与新发行的5%债券具备竞争力。反之，若利率降至2%，你那只3%的债券就变得非常抢手，买家会竞相抬高其价格。

```
反向关系：债券价格 vs. 收益率

  债券                                         债券
  价格                                         收益率
  （美元）                                     （%）
    |                                            |
1200|*                                           |            *  7%
    |  *                                         |          *
1100|    *                                       |        *    6%
    |      *                                     |      *
1000|--------*--- 票面价值 ------           5% --|----*--------
    |          *                                 |  *
 900|            *                            4%-|*
    |              *                             |
 800|                *                        3%-|
    |                  *                         |
    +---|---|---|---|---|---                      +---|---|---|---
        3%  4%  5%  6%  7%                       800  900  1000  1100
          市场收益率                                   债券价格

  收益率上升   ------>  价格下跌               镜像关系：
  收益率下降   ------>  价格上涨               同一关系的两面
```

这种反向关系源于竞争机制。债券本质上是一系列未来现金流。当市场利率变化时，这些未来现金流的现值也随之改变。利率越高，未来现金流的今日价值越低（价格下跌）；利率越低，未来现金流的今日价值越高（价格上涨）。

#### 久期：你的债券有多敏感？

并非所有债券对利率变化的反应都相同。对于相同幅度的利率变动，30年期债券的价格波动将远大于2年期债券。这种敏感性由一个叫做**久期**的概念来衡量。

久期告诉你，每当利率变动1%，债券价格将大约变动多少。

```
久期与利率敏感性

  债券A：2年期国债，久期 = 1.9年
  债券B：10年期国债，久期 = 8.5年
  债券C：30年期国债，久期 = 20.2年

  若利率上升1%：
  +---------------------------------------------------------------+
  |  债券    |  久期  |  价格变动  |  10,000美元投资的损失金额    |
  +---------------------------------------------------------------+
  |  2年期   |  1.9   |   -1.9%    |   -190美元                  |
  |  10年期  |  8.5   |   -8.5%    |   -850美元                  |
  |  30年期  |  20.2  |  -20.2%    |  -2,020美元                 |
  +---------------------------------------------------------------+

  直观示意（利率上升1%）：

  2年期：   [=========-]     价格小幅下跌
  10年期：  [====------]     价格中幅下跌
  30年期：  [--________]     价格大幅下跌

  经验法则：久期 ≈ 利率每变动1%时的近似价格变动百分比。
```

这就是为什么短期债券被认为比长期债券更安全——它们对利率变化的敏感度更低。如果你担心利率上升，就缩短久期；如果你预期利率将下降，更长久期的债券将带来更大的价格涨幅。

#### 债券的类型

债券种类繁多。了解主要类别，有助于你为投资组合选择合适的品种。

**政府债券（主权债务）：**

```
+-------------------------------------------------------------------+
|                     美国政府债券                                   |
+-------------------------------------------------------------------+
|  国库券（T-Bills）：                                              |
|    期限：4周至1年                                                 |
|    票息：无（折价出售，到期以面值偿还）                           |
|    风险：基本为零（由美国政府背书）                               |
|    示例：以980美元购入，到期收回1,000美元                        |
|                                                                   |
|  中期国债（T-Notes）：                                           |
|    期限：2至10年                                                  |
|    票息：半年付息，固定支付                                       |
|    风险：极低，但存在一定的利率变动价格风险                       |
|    最受关注：10年期国债                                           |
|                                                                   |
|  长期国债（T-Bonds）：                                           |
|    期限：20至30年                                                 |
|    票息：半年付息，固定支付                                       |
|    风险：信用风险低，但存在显著利率风险                           |
|                                                                   |
|  通胀保值国债（TIPS）：                                          |
|    期限：5年、10年或30年                                         |
|    特点：本金随通胀（CPI）调整                                   |
|    风险：抵御通胀，但仍存在利率风险                               |
|                                                                   |
|  I类储蓄债券（Series I Savings Bonds）：                         |
|    期限：至少持有1年，持有不足5年有罚息                          |
|    利率：固定利率 + 通胀调整                                     |
|    限额：每人每年1万美元                                         |
+-------------------------------------------------------------------+
```

美国国债被视为全球最安全的投资，因为它们由美国政府的完全信誉和信用背书。10年期国债收益率是全球金融领域最重要的单一数字，是抵押贷款利率、企业融资成本及几乎所有资产估值所用折现率的基准。

**公司债券：**

企业发行债券以筹集资本。公司债券的收益率高于政府债券，因为它们承载信用风险——即企业可能无法偿还债务的可能性。

```
公司债券风险谱系

  风险较低 / 收益率较低 -----------------> 风险较高 / 收益率较高

  [投资级]                                   [高收益 / "垃圾级"]
  AAA  AA  A  BBB                            BB  B  CCC  CC  C  D
  |    |   |   |                              |   |   |    |   |  |
  v    v   v   v                              v   v   v    v   v  v
 MSFT  GE  T  福特                           AMC  ?   ?    ?   ?  违约

  相对国债的收益率利差（约估）：
  AAA: +0.5%    投资级：资产负债表稳健、
  AA:  +0.7%    现金流稳定的企业。违约风险
  A:   +1.0%    较低。BBB为投资级的最低档。
  BBB: +1.5%
  --------      ------------------------------------------
  BB:  +2.5%    高收益（垃圾级）：财务状况
  B:   +4.0%    较弱或杠杆较高的企业。违约
  CCC: +7.0%    风险较高，以更高收益率作为
  CC:  +10.0%   补偿。对大多数投资者而言，
  C/D: 违约      不宜作为核心持仓。
```

#### 信用评级：债券的成绩单

信用评级机构（穆迪、标普、惠誉）评估债券发行方的信用状况并给出评级。这些评级直接影响发行方须支付的利率。

```
+-------------------------------------------------------------------+
|                      信用评级体系                                  |
+-------------------------------------------------------------------+
|  标普/惠誉     |  穆迪     |  级别          |  含义              |
+-------------------------------------------------------------------+
|  AAA           |  Aaa      |  投资级        |  最高质量          |
|  AA+/AA/AA-    |  Aa1-Aa3  |  投资级        |  高质量            |
|  A+/A/A-       |  A1-A3    |  投资级        |  中上质量          |
|  BBB+/BBB/BBB- |  Baa1-Baa3|  投资级        |  中等（投资级最低）|
|----------------|-----------|----------------|---------------------|
|  BB+/BB/BB-    |  Ba1-Ba3  |  高收益级      |  投机性            |
|  B+/B/B-       |  B1-B3    |  高收益级      |  高度投机性        |
|  CCC           |  Caa      |  高收益级      |  风险较大          |
|  CC            |  Ca       |  高收益级      |  风险极高          |
|  C             |  C        |  高收益级      |  濒临违约          |
|  D             |  ---      |  违约          |  已发生违约        |
+-------------------------------------------------------------------+

关键分界线：BBB- / Baa3是投资级与垃圾级的分水岭。
许多机构投资者被要求只持有投资级债券。
当一只债券从BBB-被下调至BB+时，就成为"堕落天使"，
被机构强制抛售可能引发价格急剧下跌。
```

#### 债券价格在实践中的运动规律

理解推动债券价格变化的力量，有助于你做出更好的投资决策。

```
影响债券价格的因素

+-------------------------------+------------------+------------------+
| 因素                          | 价格影响         | 收益率影响       |
+-------------------------------+------------------+------------------+
| 利率上升                       | 价格下跌         | 收益率上升       |
| 利率下降                       | 价格上涨         | 收益率下降       |
| 信用评级上调                   | 价格上涨         | 收益率下降       |
| 信用评级下调                   | 价格下跌         | 收益率上升       |
| 通胀预期上升                   | 价格下跌         | 收益率上升       |
| 经济衰退担忧                   | 政府债券上涨     | 政府债券收益率下降|
|                               | 公司债券下跌     | 公司债券收益率上升|
| 避险情绪升温                   | 国债上涨         | 国债收益率下降   |
+-------------------------------+------------------+------------------+
```

注意最后两行。在经济承压时期，政府债券与公司债券走势相反。投资者抛售公司债券（担忧违约），转而买入政府债券（寻求安全）。这种现象被称为"避险情绪"，也是政府债券作为投资组合减震器的核心原因之一。

#### 收益率曲线：它在告诉你什么

收益率曲线是在某一时点，展示不同期限政府债券收益率的图表。其形态能够揭示市场对经济和利率走势的大量预期。

```
收益率曲线形态

正常型（向上倾斜）：              倒挂型：
经济健康                           衰退预警信号

收益率                             收益率
  |              ****               |  ****
  |          ****                   |      ****
  |      ****                      |          ****
  |  ****                          |              ****
  |**                              |                  ****
  +---|---|---|---|---              +---|---|---|---|---
     1年  2年  5年 10年 30年           1年  2年  5年 10年 30年
```

**正常型收益率曲线：** 期限越长的债券收益率越高。这在直觉上讲得通：把资金锁定30年，你理应比锁定1年获得更高的补偿。这是最常见的曲线形态，反映出市场对经济持续增长的预期。

**倒挂型收益率曲线：** 短期收益率高于长期收益率。这种情况并不常见，历史上一直是最可靠的衰退预测信号之一。收益率曲线倒挂意味着市场预期美联储未来将因经济疲软而不得不下调短期利率。

**平坦型收益率曲线：** 所有期限的债券收益率大致相同。这通常出现在正常型与倒挂型曲线的过渡期间，往往反映出市场对经济前景的不确定——市场无法判断前方是增长还是衰退。

**陡峭型收益率曲线：** 短期与长期收益率之间的差距较大。这通常出现在经济复苏期间，彼时美联储将短期利率维持在低位，而市场对增长与通胀的长期预期推动长端收益率走高。陡峭型曲线通常对银行（借短贷长）和风险资产整体有利。

值得注意的是，收益率曲线分析并非精确的择时工具。曲线倒挂后，可能数月后才出现衰退，历史上倒挂与衰退之间的滞后期从6个月到24个月不等。但作为方向性指标，它在经济学领域有着最佳的历史记录之一。

### c) 常见误区

**误区一："债券完全没有风险。"**

债券通常比股票更安全，但并非没有风险。债券承载利率风险（利率上升时价格下跌）、信用风险（发行方可能违约）、通胀风险（固定支付随时间贬值）以及流动性风险（部分债券难以快速出售）。2022年，美国债券市场经历了历史上最惨烈的年份之一，综合债券指数下跌超过13%，原因是美联储大幅加息。"比股票更安全"并不等于"安全无虞"。

**误区二："利率上升时，我应该卖出债券。"**

不一定。如果你持有单只债券至到期，利率变化并不影响你实际收到的现金——你仍然会收到票息和面值本金。只有当你需要在到期前出售，或持有没有到期日的债券基金时，利率上升引发的价格变动才会对你造成实际影响。此外，利率上升意味着新债券支付更高的收益率，这对你再投资票息或向投资组合中添加新债券是有利的。

**误区三："高收益债券是提升收入的好方法。"**

高收益（垃圾级）债券提供更高的票息，但同时承载显著更高的违约风险。经济衰退期间，垃圾债券的违约率可飙升至10%甚至更高。更糟糕的是，高收益债券在市场承压时往往与股票正相关，这意味着它们恰恰在你最需要分散化保护的时候失灵。与投资级政府债券相比，它们根本无法提供同等的减震效果。从风险行为来看，不妨把高收益债券理解为更像股票，而非债券。

**误区四："债券基金和单只债券是一回事。"**

两者有根本区别。单只债券有明确的到期日，如果你持有至到期，你能够清楚知道自己将获得多少。债券基金永远不会到期。基金经理持续买卖债券，基金的净资产价值（NAV）每日波动。在利率上升的环境中，债券基金可能连年亏损，因为其持仓是按市值持续重新计价的。两者并无优劣之分，它们服务于不同的目的。

**误区五："票息率代表你的实际收益。"**

票息率只告诉你按面值计算的年息支付金额。如果你以溢价（高于面值）买入债券，你的实际收益（收益率）将低于票息率；如果以折价买入，收益率将高于票息率。评估债券回报时，请始终以到期收益率为准，而非票息率。

**误区六："既然股票长期回报更高，债券就是浪费钱。"**

这种说法忽视了波动性在实际投资中的关键作用。100%股票投资组合在纸面上预期回报更高，但剧烈的回撤足以让投资者在最糟糕的时机恐慌性抛售。研究一再表明，投资者行为（在恐慌中卖出，在亢奋中买入）造成的财富损失，远远超过次优资产配置所带来的损失。投资组合中的债券，不仅是为了追求回报，更是为了防止你对其余资产做出灾难性的情绪化决策。

**误区七："所有政府债券都同样安全。"**

美国国债极为安全，但其他国家的政府债券风险各异。2010年至2012年欧洲债务危机期间，希腊政府债券损失了约75%的价值。阿根廷已多次对政府债券发生违约。即便在"安全"的类别中，长期国债也承载显著的利率风险，正如2022年所展现的那样。当人们说"政府债券安全"时，通常特指短至中期的美国国债。

### d) 常见问题与解答

**问1：我的投资组合中应配置多少债券？**

答：一个常见的经验法则是，按年龄持有相应比例的债券（30岁 = 持有30%债券）。不过，许多顾问建议年轻投资者采用更进取的"年龄减20"法则（30岁 = 持有10%债券）。合适的比例取决于你的风险承受能力、投资期限、收入需求及其他财务保障来源。至少，如果你有任何在未来5年内不能承受损失的资金，那笔钱应放在债券或现金中，而非股票。

**问2：为什么2022年债券亏损如此严重？**

答：美联储以40年来最快的速度加息，在短短一年内将利率从接近0%提高到4%以上。由于债券价格与收益率反向运动，这轮急速加息引发了历史性的债券价格下跌。长久期债券受冲击最大——TLT交易所交易基金（20年以上期限国债）下跌约31%。这是一个由异常起点（接近零的利率）叠加现代史上最快紧缩周期所驱动的特殊年份。

**问3：国债与市政债券有什么区别？**

答：国债由联邦政府发行，利息免征州税和地方税，但须缴纳联邦税。市政债券（市政债）由州和地方政府发行，利息通常免征联邦税，若你居住在发行州，往往也免征州税。对于税率较高的投资者，市政债的税收优惠可能使其税后回报与国债相当甚至更优，即便其税前收益率较低。

**问4：持有政府债券会亏损吗？**

答：美国国债不会因违约而亏损（政府可以印钱偿债）。但如果你在收益率上升后于到期前出售，价格将低于购入价，从而产生亏损。此外，如果通胀超过债券收益率，你的实际购买力也会缩水。从实际（通胀调整后）收益来看，政府债券确实可能、也确实会产生负回报。

**问5：衰退期间债券表现如何？**

答：政府债券通常在衰退期间表现良好，原因是美联储降息（推动债券价格上涨），同时投资者涌向政府债务避险。公司债券，尤其是高收益债券，在衰退期间往往表现欠佳，因为违约担忧会推高信用利差。这种分化，正是政府债券作为投资组合减震器的核心价值所在。

**问6：我应该买单只债券还是债券基金？**

答：对于大多数投资规模在50万美元以下的投资者，债券交易所交易基金提供了分散化、低成本和流动性的最佳组合。如果你有特定的负债需要匹配（例如5年后支付一笔已知的支出），单只债券或债券梯形组合可能更合适，因为持有至到期可以消除利率风险。债券梯形组合是指购买在连续年份到期的债券（1年、2年、3年等），使你始终有债券到期以提供流动性。

**问7：什么是债券梯形组合？**

答：债券梯形组合是一种错开债券到期日进行持仓的策略。例如，你可以购买分别在1年、2年、3年、4年和5年后到期的债券。当1年期债券到期时，你将所得再投资于新的5年期债券，维持梯形结构。这种方式提供定期流动性，降低利率风险（你不会被锁定在某一利率水平太久），并能享受通常较高的长期收益率。

```
债券梯形组合示例（投资5万美元）

第1年：10,000美元买1年期债券  ---> 到期，再投资于5年期
第2年：10,000美元买2年期债券  ---> 到期，再投资于5年期
第3年：10,000美元买3年期债券  ---> 到期，再投资于5年期
第4年：10,000美元买4年期债券  ---> 到期，再投资于5年期
第5年：10,000美元买5年期债券  ---> 到期，再投资于5年期

        第1年     第2年     第3年     第4年     第5年
        [债券A]  [债券B]  [债券C]  [债券D]  [债券E]
                 [新E1]  [债券C]  [债券D]  [债券E]
                          [新E2]  [债券D]  [债券E]

每年：一只债券到期，提供流动性
每次再投资：锁定当时市场利率
```

**问8：什么是"无风险利率"，为什么以债券为基准？**

答：无风险利率是你在零违约风险下理论上能获得的回报。实践中，短期美国国库券的收益率被用作无风险利率，因为美国政府从未对其债务发生违约。这一利率是整个金融体系的基础，因为每项投资的预期回报都被定义为无风险利率加上风险溢价。当无风险利率变化时，从股票到房地产到公司债券的所有资产都会重新定价。

**问9：什么是通胀保值国债（TIPS），我应该持有吗？**

答：通胀保值国债是本金随通胀（以CPI衡量）调整的政府债券。若通胀率为3%，面值1,000美元的通胀保值国债将调整至1,030美元，票息按调整后的更高本金计算。通胀保值国债能抵御通胀，但当实际利率上升时也可能表现不佳。对于特别担心通胀侵蚀购买力的投资者而言，它是有用的配置品种，但不应替代分散化投资组合中的普通国债。

**问10：利率上升如何影响我的股票投资组合？**

答：债券收益率上升会提高对未来现金流进行折现的折现率，从而压低股价，尤其是那些价值高度依赖遥远未来盈利的成长股。利率上升也使债券相对股票更具吸引力，部分资金会从股市流向债市。然而，如果利率上升是经济强劲的结果，企业盈利的增长有可能抵消折现率上升的负面影响。利率上升的速度和幅度，比绝对水平本身更为关键。

**Stella：** 这就是债券比股票更安全的原因？

**Horace：** 这是原因之一。另一个原因是债券支付的合同性质。买股票，公司可能发股息，也可能不发，随时可以削减。没有人保证你的回报。债券是法律合同，发行方在法律上有义务按时支付每一笔票息，并在到期日归还本金。如果他们错过一笔付款，就构成技术性违约，债券持有人可以采取法律行动。这种合同确定性，是股票根本无法提供的。

[VISUAL: A line of people at a window labeled "公司破产——谁先拿到钱？" The order from front to back: "有担保债券持有人"，"无担保债券持有人"，"优先股股东"，"普通股股东" (with a sad face). Text: "债券 = 贷款方优先权"]

**Stella：** 所以债券持有人先拿钱，所以更安全？

**Horace：** 一般来说是的。还有一层保障。债券支付是合同义务。苹果说他们要支付3%的利息，这是法律承诺。如果他们违约，债券持有人可以强制公司进入破产程序。股票股息则不同，董事会随时可以削减或取消，没有人有义务必须支付。

**Stella：** 明白了。那带我走一遍基本结构。债券的组成部分是什么？

**Horace：** 每只债券都有三个关键要素。第一是面值，也叫票面价值，就是你出借的金额，通常是1,000美元。第二是票息率，也就是年利率。1,000美元债券的4%票息，意味着每年40美元。第三是到期日，就是发行方归还你1,000美元的那天。

[ANIMATION: Reference animation/week05_bond_pricing.py - An animation showing a bond being constructed piece by piece. A $1,000 bill appears (面值), then a percentage sign drops in (票息率 = 4%), then a calendar flips to show a date 10 years out (到期日). Cash flow arrows then animate: small $20 arrows flowing from the bond to the investor every 6 months, and a large $1,000 arrow flowing back at the end.]

**Stella：** 等等，你说每年40美元，但付款是每六个月一次？

**Horace：** 观察得很仔细。大多数债券每半年付息一次。所以4%票息意味着两笔各20美元的付款，间隔六个月。到债券到期时，你收到最后一笔票息加上1,000美元面值本金。

**Stella：** 所以如果我买了一只10年期、票息率4%的债券，我就能知道我会拿到什么：每六个月20美元，持续10年，最后再拿回1,000美元？

**Horace：** 如果你持有至到期，且发行方不违约，是的。这种可预期性是债券最吸引人的地方之一。股票的回报永远是未知数，而债券从第一天起就能让你算清楚每一笔现金流。

[VISUAL: Side-by-side comparison. Left: "股票收益" showing a jagged, unpredictable line with question marks. Right: "债券现金流" showing a clean, predictable staircase of coupon payments with a final step up for the principal return. Text: "债券 = 可预期的现金流"]

**Stella：** 好，这里是我一直搞不清楚的地方。你说票息率是4%，但我也听到人们说"收益率"。这两个是不同的东西吗？

**Horace：** 这是债券投资中最重要的区别之一，几乎所有人一开始都会混淆。票息率是固定的，永远不变，可以说是印在债券上的。但债券发行后可以在二级市场交易，价格可能高于或低于1,000美元。

**Stella：** 价格为什么会变？

**Horace：** 因为利率。假设你买了一只票息3%的债券。一年后，市场上新发行的债券票息是5%。这时你那只3%的债券就显得不那么吸引人了。如果你想出售，没有人愿意为一只3%的债券付全价，因为他们可以直接买新发行的5%债券。所以你的债券价格必须下跌。

**Stella：** 这说得通。有更好的选择，旧的就必须便宜一点。

**Horace：** 对。而价格下跌，收益率就上升了。核心公式是这样的：当前收益率等于年票息除以当前价格。你那只3%票息的债券每年支付30美元。如果价格跌到900美元，收益率就是30除以900，等于3.33%。债券的收益率上升了，变得更有竞争力。

[VISUAL: An animated scale/balance. On one side: "债券价格" with a weight. On the other side: "债券收益率" with a weight. As 债券价格 goes down, 债券收益率 goes up, and vice versa. Text in center: "永远相反，绝无例外。"]

**Stella：** 所以价格和收益率朝相反方向运动？

**Horace：** 永远如此。这是债券的铁律，不是某种趋势或相关性，而是数学上的必然。价格涨，收益率跌；价格跌，收益率涨。如果今天你只记住一件事，记住这个。

**Stella：** 为什么这一点这么重要？

**Horace：** 因为它直接影响你的投资组合。美联储加息，债券价格就跌。美联储降息，债券价格就涨。2022年，美联储以40年来最快的速度加息，债券市场经历了历史上最惨烈的年份之一。综合债券市场指数下跌超过13%，长期国债下跌约31%。

**Stella：** 等等，债券不是应该很安全吗！

**Horace：** 这就是微妙之处。债券在到期收回本金方面是安全的，但在持有期间价格完全可能大幅波动，尤其是长期债券。如果利率急剧上升，30年期国债单年可能损失20%。短期债券，比如1至2年期国债，就稳定得多，因为其价格几乎不动。

[VISUAL: Three horizontal bars showing price sensitivity. "2年期债券：价格波动小" (thin bar, small left-right movement). "10年期债券：价格波动中等" (medium bar, larger movement). "30年期债券：价格波动大" (long bar, dramatic movement). Text: "久期 = 对利率变化的敏感度"]

**Stella：** 你刚才说了一个重要的点——期限越长越敏感。这个有没有量化指标？

**Horace：** 有，叫做久期。久期告诉你，利率每变动1%，债券价格大约变动多少。久期为5年的债券，如果利率上升1%，价格约下跌5%；久期为20年的债券，价格约下跌20%。

**Stella：** 那如果我担心利率上升，我就应该选短久期？

**Horace：** 正是。短久期债券是防守性选择。如果你认为利率将要下降，长久期债券则能给你带来更大的涨幅。这是债券投资中一个关键的战略决策。

**Stella：** 我们来聊聊债券的种类。我知道有政府债券和公司债券，真正的区别是什么？

**Horace：** 最大的区别是信用风险。美国国债由美国政府的完全信誉和信用背书，是全球公认最安全的投资，因为政府可以征税，甚至可以印钱来偿债。美国国债违约的可能性基本为零。

**Stella：** 公司债券呢？

**Horace：** 公司债券由企业发行，企业可以而且确实会破产。所以公司债券承载信用风险——发行方可能无力偿还的风险。为了补偿你承担的风险，公司债券支付比国债更高的收益率。高出国债的那部分额外收益就叫做信用利差。

[VISUAL: A staircase/layer diagram. Bottom step: "美国国债：收益率4.0%，违约风险基本为零". Second step: "投资级公司债（AAA-BBB）：收益率5.0-5.5%，违约风险低". Top step: "高收益/垃圾级（BB及以下）：收益率7.0-9.0%以上，违约风险较高". Arrows showing the spread between each level.]

**Stella：** 我怎么知道哪些公司风险大，哪些风险小？

**Horace：** 这就是信用评级的用途。标普、穆迪、惠誉这三家主要评级机构会评估每个债券发行方，并给出字母评级。AAA是最高级别，代表极度安全；依次往下是AA、A、BBB，其中BBB-是所谓投资级的最低一档。

**Stella：** 再往下呢？

**Horace：** 低于投资级就是高收益级，也叫垃圾债券，评级为BB及以下。这类债券支付更高的收益率，因为企业真实存在违约的可能性。正常时期，垃圾债券的违约率约为2%至3%；经济衰退期间，可能飙升到10%甚至更高。

**Stella：** 所以垃圾债券并不是减震器。

**Horace：** 完全不是。事实上，危机时期垃圾债券的走势更像股票。经济走弱时，投资者担忧违约，抛售垃圾债券，其价格在股票同步崩跌时也一起跌。这正是你投资组合中债券部分最不想看到的结果。

[VISUAL: Two-panel comparison during "2008年金融危机。" Left panel: "国债" showing a green upward arrow (+20%). Right panel: "垃圾债券" showing a red downward arrow (-26%). Text: "并非所有债券生来平等"]

**Stella：** 这个区分非常重要。所以用来做减震器，我要的是政府债券，而不是垃圾债券。

**Horace：** 正是。政府债券和投资级债券才是你的减震器。垃圾债券是另一种工具，服务于不同目的，更偏向增强收益，但在下行期间保护不了你。

**Stella：** 我们来聊聊收益率曲线。新闻里经常提到，尤其是"倒挂"的时候。它到底是什么？

**Horace：** 收益率曲线其实就是一张图，展示某个时点上，不同期限政府债券收益率的分布。左边是短期债券，比如3个月和2年期国债；右边是长期债券，比如10年期和30年期国债。正常情况下，曲线向上倾斜，因为期限越长的债券需要支付更高的收益率，以补偿额外的时间和不确定性。

**Stella：** 说得通。为什么我要接受把钱锁定30年却跟锁定1年拿一样的收益呢？

**Horace：** 当然不会接受。正因如此，向上倾斜的正常型收益率曲线才是常态。但有时曲线会倒挂，意味着短期收益率高于长期收益率，这就是大事了。

[ANIMATION: Reference animation/week05_bond_pricing.py - An animated yield curve that morphs between shapes. Starts with a normal upward slope, then gradually flattens, then inverts. Labels appear at each stage: "正常型：经济健康", "平坦型：过渡时期", "倒挂型：衰退预警". A historical timeline below shows previous inversions followed by recessions, with dates.]

**Stella：** 为什么收益率曲线倒挂能预测衰退？

**Horace：** 想想收益率曲线代表什么。长期收益率反映了市场对未来短期利率的预期。如果市场预期经济将放缓，就预期美联储未来必须降息。于是长期收益率跌破当前短期收益率。历史上，过去50年美国每一次经济衰退之前，都出现过收益率曲线倒挂。它不是精确的择时工具，但却是我们拥有的最可靠的预警信号之一。

**Stella：** 太有意思了。那我应该怎么把债券用在我的投资组合里？

**Horace：** 经典做法是60/40投资组合：60%股票，40%债券。我来告诉你为什么这行得通。根据历史数据，100%股票投资组合年化收益约为10%，但伴随着让人痛不欲生的回撤——最差年份跌了37%。60/40投资组合年化收益约为8.5%，每年只少了约1.5%，但最差年份仅跌了20%，波动性也从约15%降至10%以下。

**Stella：** 所以你放弃一点收益，换来更大的稳定性。

**Horace：** 这就是这笔交换。而且大多数人忽视了一点：稳定不仅仅是为了舒适，更是为了行为管理。当你的投资组合跌了37%，把一切清仓换成现金的冲动几乎无法抵抗。2008年和2009年，有数百万人正是这样做的——在最低点锁定了亏损，然后错过了反弹。但当你的投资组合只跌了20%，虽然依然痛苦，大多数人还是撑得过去。投资组合里的债券，保护你的不只是资产，还有你不去犯那些最灾难性的情绪化错误。

[VISUAL: Two investor reactions side by side. Left: "2008年的100%股票投资者" showing a panicking figure selling at the bottom (-37%), then the recovery happens without them. Right: "2008年的60/40投资者" showing a concerned but calm figure holding through -20%, then participating in the recovery. Final wealth comparison shown below.]

**Stella：** 这一点太关键了。最好的投资组合，是你真正能坚持持有的那个。

**Horace：** 完全正确。一个理论上最优但你在恐慌时抛弃的投资组合，远比一个稍次优但你能坚守的投资组合糟糕得多。

**Stella：** 那我实际上应该怎么投资债券？买单只债券，还是用基金？

**Horace：** 对大多数人来说，债券交易所交易基金是最佳选择。像BND或AGG这样的交易所交易基金，能让你以约0.03%的年费持有数千只债券，涵盖政府债和公司债、各个期限。每100美元收费三分钱，分散化极好，成本极低，像普通股票一样买卖。

**Stella：** 直接向政府买国债呢？

**Horace：** 你可以通过美国政府官网TreasuryDirect.gov直接购买国债。这对于I类储蓄债券尤其值得考虑——这种抗通胀储蓄债券每人每年限购1万美元。其他国债的话，通过券商购买通常更方便。

[VISUAL: A comparison table showing four ways to buy bonds: "债券交易所交易基金" (BND, AGG, TLT), "单只债券" (通过券商), "TreasuryDirect" (I类债券, 国库券), and "债券共同基金" (先锋, 富达). Each with a brief pro and con listed.]

**Stella：** 债券梯形组合呢？我听说过这个词。

**Horace：** 债券梯形组合是一种购买在连续年份到期债券的策略。比如你分别购买1年、2年、3年、4年、5年后到期的债券。每年，期限最短的那只到期，你收回本金，再投资买一只新的5年期债券。这样可以获得定期流动性，降低利率风险，长期下来，你的实际收益率会趋近于这些年份利率的平均值。

**Stella：** 这听起来很实用。

**Horace：** 确实如此。对于需要定期收入的退休人员尤其有用——每年有一只债券到期，提供生活所需的现金，而梯形组合的其余部分继续赚取利息。

**Stella：** 说到实用，我问一下市政债券。我听说它有税收优惠？

**Horace：** 市政债券，也叫市政债，由州和地方政府发行。最大的优势是其利息通常免征联邦所得税，如果你居住在发行州，往往也免征州税。对于税率较高的人——比如联邦税率35%——一只收益率3.5%的市政债，相当于一只税前收益率约5.4%的应税债券。这种税收优惠在数字上非常可观。

**Stella：** 所以市政债更适合高税率人群？

**Horace：** 一般来说是的。如果你在10%或12%的税率档，税收优惠很小，通常还是普通国债或公司债的税前收益率更高。但如果你在32%或更高的税率档，市政债值得认真考虑，尤其是在应税账户中。在个人退休账户这类税收优惠账户里，税收好处不适用，市政债的吸引力就没那么大了。

[VISUAL: A side-by-side tax comparison. "应税债券：5%收益率，税率35%，税后：3.25%". "市政债券：3.5%收益率，免税，税后：3.5%". The muni wins despite the lower headline yield. Text: "永远以税后收益率进行比较"]

**Stella：** 这个关于应税账户和税收优惠账户区别的提示很重要。好，有件事最近一直让我困惑。2022年，股票和债券同时下跌，这不应该发生的，对吗？

**Horace：** 问得好。历史上，股票和债券在危机中往往反向运动——股票跌，债券涨，投资组合保持相对稳定。但2022年是特殊情况，主要驱动力是通胀和美联储的激进加息。通胀同时打击股票（通过抬高成本和折现率）和债券（通过推高收益率）。当通胀是主要风险时，正常的股债分散化效果可能失灵。

**Stella：** 那是不是说60/40已经过时了？

**Horace：** 不是。2022年是例外，不是常态。股债分散化的效果在过去大半个世纪里都是有效的。一个坏年份推翻不了几十年的历史证据。重要的是，造成2022年局面的条件——接近零的利率叠加突发性通胀冲击——在历史上极为罕见。如今债券收益率已回归较为正常的水平，未来若经济衰退成为主要风险而非通胀，分散化效果理应如预期般发挥作用。

[VISUAL: Timeline showing stock-bond correlation. Most periods show negative correlation (bonds rally when stocks fall), with 2022 highlighted as an exception. Text: "2022年是例外，而非新常态"]

**Stella：** 通胀保值国债呢？TIPS？

**Horace：** 通胀保值国债是一种特殊的国债，本金随通胀调整。如果通胀率为3%，1,000美元的通胀保值国债本金调整至1,030美元，票息按调整后的更高本金计算。对于担心通胀侵蚀购买力的投资者来说很有用，但仍然存在利率风险，如果实际利率上升，其价格同样可能下跌。

**Stella：** 最后一个问题。大家说的"10年期收益率"，为什么这个数字这么重要？

**Horace：** 10年期国债收益率是整个金融领域最受关注的单一数字。它是抵押贷款利率、汽车贷款、企业融资成本以及全球几乎所有资产估值所用折现率的基准。10年期收益率上升，整个经济的借贷成本就提高：房贷利率上涨，企业扩张放缓，股票估值承压。收益率下降，则反之。如果你想了解经济走向，就盯着10年期收益率。

**Stella：** 这个背景信息真的很有帮助。让我来总结一下：债券是贷款，你获得可预期的利息支付；价格与收益率永远朝相反方向运动；政府债券最安全，危机保护效果最好；久期衡量债券对利率变化的敏感度；信用评级告诉你公司债券有多大风险；债券在投资组合中扮演关键的减震器角色，不仅保护你的资产，也保护你不犯情绪化的错误。

**Horace：** 总结得非常好。核心结论是：债券或许无聊，但无聊是它的特性，而非缺陷。你的投资组合需要一个锚，债券就是那个锚。

**Stella：** 说实话，这比我预想的要有意思得多。价格与收益率的反向关系，还有收益率曲线这些，真的很迷人。

**Horace：** 一旦你开始关注债券市场，就会发现它和经济中的一切都息息相关。就像学会了读一门新语言。

**Stella：** 下周讲什么？

**Horace：** 下周我们要聊黄金和大宗商品。同样是在投资组合中发挥分散化作用的资产类别，但背后的逻辑与债券截然不同。

**Stella：** 走之前，快问快答？

**Horace：** 来吧。

**Stella：** 持有政府债券有可能亏损吗？

**Horace：** 有，如果你在利率上升后于到期前出售，价格会低于购入价。另外，如果通胀超过你的收益率，购买力也会缩水。但美国国债不会因违约而亏损。

**Stella：** 投资债券的最低门槛是多少？

**Horace：** 单只债券通常最低1,000美元。BND这样的债券交易所交易基金买一股就行，大约七八十美元左右。部分券商支持碎股交易，1美元也能起步。

**Stella：** 初学者最推荐的债券？

**Horace：** 总体债券市场交易所交易基金，比如BND或AGG。一步到位的分散化，超低费率，任何券商账户都能买到。

**Stella：** 应该在利率高的时候还是低的时候买债券？

**Horace：** 利率高的时候其实是买债券的好时机，因为你锁定了更高的收益率。利率低的时候，债券收益率没有吸引力，而且利率一旦上升价格就会下跌。最不该大量买债券的时候，正是2020至2021年利率接近零的阶段。

**Stella：** 债券投资者最容易犯的错误是什么？

**Horace：** 盲目追求高收益率。仅仅因为支付更高票息，就买入高风险的垃圾债券，却不了解其中的信用风险。债券在你投资组合中的作用是提供稳定性，而不是最大化收入。如果你想要更高回报，提高股票配置比例，而不是去买垃圾债券。

**Stella：** I类储蓄债券呢？我听说几年前很火。

**Horace：** I类储蓄债券是通过TreasuryDirect.gov直接从美国政府购买的储蓄债券。其利率由固定利率加上每半年调整一次的通胀补贴构成。2022年通胀率高达9%时，I类债券的利率超过了9%，吸引了大批关注。每人每年限购1万美元，至少持有1年，持有不足5年卖出有三个月利息的罚金。在可申购额度内，这是一个优质的低风险抗通胀储蓄工具。

**Stella：** 最后一个：我怎么知道该选短期债券还是长期债券？

**Horace：** 把债券久期与你的投资期限和市场判断结合起来考虑。如果你2至3年内需要用到这笔钱，选短期债券或SHY这样的短期债券交易所交易基金，非常稳定。如果你期限较长，且认为利率将下行，TLT这样的长期债券可以带来收入和资本利得双重回报。如果你没有明确的时间节点，也对利率走势没有特定判断，BND或AGG这样的中期综合基金能提供均衡的中间选择。
